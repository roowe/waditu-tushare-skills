# 获取实时行情

> **问题**：怎么获取实时行情？

## 回答

使用 `rt_k`（实时日K线）和 `rt_min`（实时分钟线）两个接口组合获取实时行情数据。

### rt_k — 实时日K线

获取全市场股票的实时日K线数据，支持全部股票同时返回。

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 获取全市场实时日K线
df = pro.rt_k(trade_date='20260509')
print(df.head())
```

**使用场景**：
- 盘中实时监控全市场涨跌
- 尾盘选股策略
- 每日行情快照

### rt_min — 实时分钟行情

获取单只股票的实时分钟级别行情，支持 1-60 分钟频率。

```python
# 获取某股票实时分钟行情
df = pro.rt_min(ts_code='000001.SZ', freq='1min')
print(df.head())
```

**使用场景**：
- 盘口级别短线交易
- 分时图分析
- 日内策略回测验证

### 注意事项

- 实时接口对积分要求较高，建议 5000 积分以上使用
- `rt_k` 返回的是当日实时日K（包含盘中数据），与收盘后的 `daily` 接口不同
- `rt_min` 的数据延迟取决于调用频率，非严格意义上的逐笔数据
