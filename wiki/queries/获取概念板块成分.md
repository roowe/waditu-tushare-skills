# 获取概念板块成分

> **问题**：怎么获取概念板块成分？

## 回答

根据数据源不同，Tushare 提供四个概念板块成分接口。不同数据源的板块定义和覆盖范围有差异，需根据需求选择。

### 接口对比

| 接口 | 数据源 | 说明 |
|------|--------|------|
| `dc_member` | 东方财富 | 覆盖面广，更新及时 |
| `ths_member` | 同花顺 | 概念分类细致，市场认可度高 |
| `tdx_member` | 通达信 | 面向技术分析用户 |
| `kpl_concept_cons` | 开盘啦 | 短线题材分类，适合打板 |

### 使用示例

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 东方财富概念板块成分
df = pro.dc_member(concept_code='BK0001')

# 同花顺概念板块成分
df = pro.ths_member(concept_code='885902')

# 通达信概念板块成分
df = pro.tdx_member(concept_code='880001')

# 开盘啦概念板块成分
df = pro.kpl_concept_cons(concept_code='BK0001')
```

### 选择建议

1. **日常研究**：优先使用 `dc_member`（东财），板块分类全面，社区讨论活跃
2. **深度分析**：`ths_member`（同花顺）概念体系成熟，研报引用频率高
3. **技术分析**：`tdx_member`（通达信）与行情软件联动方便
4. **短线交易**：`kpl_concept_cons`（开盘啦）题材分类贴近市场热点

### 注意事项

- 不同数据源的概念板块编码体系不同，不能混用
- 同一概念在不同数据源中的成分股可能有差异
- 建议选定一个主数据源，避免跨源比较
