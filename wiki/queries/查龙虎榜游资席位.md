# 查龙虎榜游资席位

> **问题**：怎么查龙虎榜游资席位？

## 回答

通过 `top_list`、`top_inst`、`hm_detail`、`hm_list` 四个接口组合查询龙虎榜和游资席位数据。

### 1. top_list — 龙虎榜每日明细

获取每日龙虎榜交易明细，包含买入/卖出席位和金额。

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 查询某日龙虎榜
df = pro.top_list(trade_date='20240115')

# 查询某股票历史上榜记录
df = pro.top_list(ts_code='000001.SZ', start_date='20240101', end_date='20240131')
```

### 2. top_inst — 龙虎榜机构成交明细

获取龙虎榜中机构专用席位的成交数据。

```python
# 某日机构席位明细
df = pro.top_inst(trade_date='20240115')
```

### 3. hm_detail — 游资交易明细

获取知名游资席位的逐笔交易记录。

```python
# 按游资名称查询
df = pro.hm_detail(name='国泰君安证券上海江苏路')
```

### 4. hm_list — 游资名录

获取全部已知游资席位列表。

```python
# 获取游资名录
df = pro.hm_list()
```

### 组合查询示例

典型的龙虎榜游资分析流程：

```python
# Step 1: 获取某日龙虎榜
top = pro.top_list(trade_date='20240115')

# Step 2: 查看机构席位
inst = pro.top_inst(trade_date='20240115')

# Step 3: 获取游资名录
hm = pro.hm_list()

# Step 4: 查询特定游资近期交易
detail = pro.hm_detail(name='国泰君安证券上海江苏路')
```

### 使用场景

- **短线跟庄**：跟踪知名游资席位买入个股
- **机构行为分析**：通过 top_inst 识别机构建仓/减仓方向
- **市场情绪**：龙虎榜上榜数量和金额反映市场活跃度
