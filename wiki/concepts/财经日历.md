# 财经日历

> 财经日历（eco_cal）提供全球经济事件的发布时间表，包括各国重要经济指标（如非农就业、CPI、PMI、GDP 等）的前值、预测值和今值。是跟踪全球宏观数据发布节奏、预判市场波动的重要工具。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 财经日历(eco_cal) | 全球经济事件发布日历 | `eco_cal` |
| 前值(pre_value) | 上期实际公布的数值 | `pre_value` |
| 预测值(fore_value) | 市场对本期数值的预期 | `fore_value` |
| 今值(value) | 本期实际公布的数值 | `value` |
| 非农就业 | 美国非农就业数据，全球最重要的经济指标之一 | event='美国季调后非农*' |
| CPI/PMI 等全球指标 | 各国通胀、景气度等指标发布 | 按国家/事件筛选 |
| 国家/货币 | 按国家或货币代码筛选事件 | `country`, `currency` |

## 详细说明

### 数据结构与获取

`eco_cal` 接口提供全球经济事件的日历数据，单次最大获取 100 行，2000 积分可调用。每条记录包含日期（date）、时间（time）、国家（country）、货币代码（currency）、事件名称（event）、今值（value）、前值（pre_value）和预测值（fore_value）。

```python
pro = ts.pro_api()

# 获取指定日期全球经济日历
df = pro.eco_cal(date='20200403')

# 获取中国经济事件
df = pro.eco_cal(country='中国')

# 获取美国非农数据
df = pro.eco_cal(event='美国季调后非农*')
```

### 核心经济指标跟踪

财经日历覆盖全球主要经济体的核心指标发布，包括但不限于：

- **美国**：非农就业人口变动、失业率、CPI/PPI、GDP、ISM 制造业/非制造业 PMI、零售销售、初请失业金等
- **中国**：CPI/PPI、官方制造业/非制造业 PMI、GDP、社融/M2、贸易帐、工业增加值等
- **欧元区/日本/英国**：CPI、GDP、PMI、利率决议等

### 前值/预测值/今值的意义

经济数据公布时，市场最关注的是今值与预测值的偏差（surprise）：
- 今值好于预测值，通常会推动相关资产价格上涨（如非农超预期 -> 美元走强）
- 今值差于预测值，则相反
- 前值用于对比数据的边际变化方向

### 筛选与过滤

支持按国家（country）、货币代码（currency）、事件名称（event，支持通配符模糊匹配）进行筛选。例如 `event='*非农*'` 可以匹配所有包含"非农"的事件，`country='中国'` 可以只看中国经济数据。
