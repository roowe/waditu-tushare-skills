# 情绪周期
> 情绪周期是描述A股市场短线投机氛围从低迷到亢奋再到衰退的循环过程，是短线交易者择时的核心框架。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 情绪周期 | Sentiment Cycle | 市场投机情绪的周期性变化，典型路径为冰点 → 修复 → 高潮 → 退潮 | - |
| 弱转强 | Weak-to-Strong | 个股在前一日表现弱势后次日超预期走强，是情绪修复的重要信号 | - |
| 分歧转一致 | Divergence-to-Consensus | 市场对某个方向从多空分歧走向一致看多，通常伴随缩量加速 | - |
| 首板/二板 | 1st/2nd Limit-Up | 个股的第一个和第二个涨停板，首板数量是衡量市场情绪的领先指标 | - |
| 抄底 | Bottom Fishing | 在市场情绪冰点时逢低买入，博弈情绪修复带来的反弹 | - |
| 逃顶 | Top Exit | 在市场情绪高潮时及时卖出，规避退潮期的系统性杀跌 | - |
| 神奇九转 | TD Sequential | 由Tom DeMark发明的技术分析序列，通过连续9根K线判断趋势衰竭和反转时机 | `stk_nineturn` |

## 详细说明

### 情绪周期的四个阶段

**冰点阶段**
- 市场特征：涨停家数极少（通常少于10家），连板高度被压制在2-3板，跌停家数增多
- 资金行为：场内资金普遍亏损，交投清淡，多数短线资金空仓观望
- 操作策略：关注逆势走强的题材和个股，为修复期做准备

**修复阶段**
- 市场特征：涨停家数开始增多，出现弱转强个股，连板高度拓展至4-5板
- 资金行为：聪明资金开始试探性建仓，龙头股走出超预期走势
- 操作策略：积极参与弱转强和分歧转一致的机会，试错新题材龙头

**高潮阶段**
- 市场特征：涨停家数激增（可达50-100家），连板高度拓展至7板以上，板块普涨
- 资金行为：各路资金蜂拥入场，龙头股加速缩量涨停，补涨股全面开花
- 操作策略：持有龙头股享受溢价，但需警惕一致转分歧的信号

**退潮阶段**
- 市场特征：高位股开始杀跌，龙头股断板，涨停家数骤降，亏钱效应扩散
- 资金行为：获利盘集中兑现，短线资金快速撤离，部分资金转向防御品种
- 操作策略：降低仓位，规避高位接力，等待冰点期的到来

### 弱转强与分歧转一致

**弱转强**是短线交易中最重要的模式之一，指个股在前一日炸板或大幅回落（表现弱势）后，次日超预期高开并快速走强封板。弱转强通常出现在情绪修复期的初期，是市场做多意愿恢复的重要信号。

**分歧转一致**指市场对某个方向从多空博弈走向一致看多。在盘面上表现为：前一日放量分歧（大量换手），次日缩量涨停（筹码锁定良好）。分歧转一致往往意味着新的上升波段开启。

### 神奇九转（TD序列）

TD序列（TD Sequential）是由Tom DeMark开发的技术分析工具，通过计数连续的上涨或下跌K线来判断趋势的衰竭点。当计数达到9时，预示当前趋势可能即将反转。

在A股短线交易中，神奇九转常用于：
- 判断连板股的高度极限
- 识别退潮期中的反弹节点
- 辅助判断冰点期的底部区域

Tushare提供 `stk_nineturn` 接口获取个股的TD九转序列数据。
