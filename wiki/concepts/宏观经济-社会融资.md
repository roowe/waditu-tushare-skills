# 宏观经济-社会融资

> 覆盖社会融资规模、新增信贷等金融数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `sf_month` | 获取月度社会融资数据 | m, start_m, end_m |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 sf_month 接口
df = pro.sf_month()
print(df.head())
```
