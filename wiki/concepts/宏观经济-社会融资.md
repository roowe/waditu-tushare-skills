# 宏观经济-社会融资

> 覆盖社会融资规模、新增信贷等金融数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `sf_month` | 获取月度社会融资数据 | m, start_m, end_m |

## 接口详细参数

### `sf_month` 详细说明

**调用说明**: 需2000积分；单次最大2000条数据，可循环提取
**数据说明**: 月度数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| m | str | N | 月份（YYYYMM），支持多个月份同时输入，逗号分隔 |
| start_m | str | N | 开始月份 |
| end_m | str | N | 结束月份 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| month | str | 月度 |
| inc_month | float | 社融增量当月值（亿元） |
| inc_cumval | float | 社融增量累计值（亿元） |
| stk_endval | float | 社融存量期末值（万亿元） |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 sf_month 接口
df = pro.sf_month()
print(df.head())
```
