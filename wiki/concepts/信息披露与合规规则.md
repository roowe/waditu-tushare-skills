> 信息披露制度是资本市场健康运行的基石，上市公司必须按规定时限和格式披露定期报告和临时报告。了解合规规则对量化投资尤为重要——减持新规、窗口期限制、短线交易禁止等规则直接影响数据的可用性和策略的合法性。

## 核心概念

| 概念 | 说明 | 关键接口/字段 |
|------|------|-------------|
| 定期报告 | 年报/半年报/季报，需在规定时限内披露 | `income`/`balancesheet`/`cashflow`(ann_date) |
| 临时报告 | 重大事项的即时公告（增减持/回购/ST等） | 各接口的 ann_date |
| 年报披露时限 | 会计年度结束后4个月内（次年4月30日前） | `disclosure_date` |
| 半年报披露时限 | 半年度结束后2个月内（8月31日前） | `disclosure_date` |
| 季报披露时限 | 季度结束后1个月内 | `disclosure_date` |
| 减持新规 | 2017年证监会《减持若干规定》，限制大股东/董监高减持节奏 | `stk_holdertrade` |
| 预披露制度 | 大股东通过集中竞价减持需提前15个交易日公告 | - |
| 减持比例限制 | 集中竞价3个月内不超过1%，大宗交易受让方6个月不得转让 | - |
| 窗口期限制 | 定期报告公告前30日内/业绩预告/快报公告前10日内，董监高不得买卖 | - |
| 短线交易禁止 | 《证券法》第47条：董监高/大股东6个月内买入又卖出(或卖出又买入)的收益归公司所有 | - |
| 关联交易 | 上市公司与关联方之间的交易，需履行审批和披露程序 | - |
| 内幕交易禁止 | 利用未公开的重大信息进行交易，构成刑事犯罪 | - |
| 敏感期交易限制 | 重大事项筹划期间的交易限制 | - |

## 详细说明

### 1. 信息披露制度

#### a. 定期报告

- 年报: 次年1月1日-4月30日（会计年度结束后4个月内）
- 一季报: 4月1日-4月30日
- 半年报: 7月1日-8月31日（半年度结束后2个月内）
- 三季报: 10月1日-10月31日
- Tushare中 `disclosure_date` 字段记录披露日期（预约日期），`ann_date` 为实际公告日期

#### b. 临时报告

重大事项发生时立即公告：

- 重大资产重组/收购/出售
- 股东增减持达到披露标准
- 股权质押/解押
- 分红派息方案
- ST/\*ST标记变更

### 2. 减持新规（2017年修订）

#### a. 适用范围

大股东（持股5%以上）/特定股东（IPO前/非公开发行获得股份的股东）/董监高

#### b. 集中竞价减持限制

- 任意连续90日内减持不超过公司股份总数的1%
- 需提前15个交易日公告减持计划

#### c. 大宗交易减持限制

- 任意连续90日内减持不超过公司股份总数的2%
- 受让方在受让后6个月内不得转让

#### d. 协议转让限制

转让后大股东持股比例不得低于5%

#### e. 不得减持的情形

涉嫌违规被立案调查/行政处罚/被交易所公开谴责未满3个月等

### 3. 窗口期与短线交易

#### a. 窗口期限制

董监高不得买卖股票的期间：

- 定期报告公告前30日内
- 业绩预告/业绩快报公告前10日内
- 自可能对股票价格产生较大影响的重大事件发生之日起或在决策过程中至依法披露后2个交易日内

#### b. 短线交易（证券法第47条）

- 上市公司董监高/持股5%以上股东
- 6个月内买入又卖出或卖出又买入
- 所得收益归公司所有
- 董事会应当收回其所得收益

### 4. 对量化研究的影响

- **数据时效性**: 减持等交易数据有公告日(ann_date)，应使用公告日而非交易发生日
- **策略合规**: 策略不应基于内幕信息（未公告的重大事项）
- **数据完整性**: 关注披露规则变化对数据连续性的影响
- **ST标记**: 使用实施日(imp_date)而非公告日(pub_date)作为ST生效时点
