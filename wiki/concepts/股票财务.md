# 股票财务

> 覆盖利润表、资产负债表、现金流量表、财务指标等上市公司财务数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `express` | 获取上市公司业绩快报 | ts_code, ann_date, start_date, end_date |
| `forecast` | 获取业绩预告数据 | ts_code, ann_date, start_date, end_date |
| `fina_mainbz` | 获得上市公司主营业务构成，分地区和产品两种方式 | ts_code, period, type, start_date |
| `dividend` | 分红送股数据 | ts_code, ann_date, record_date, ex_date |
| `income` | 获取上市公司财务利润表数据 | ts_code, ann_date, f_ann_date, start_date |
| `cashflow` | 获取上市公司现金流量表 | ts_code, ann_date, f_ann_date, start_date |
| `fina_audit` | 获取上市公司定期财务审计意见数据 | ts_code, ann_date, start_date, end_date |
| `fina_indicator` | 获取上市公司财务指标数据，为避免服务器压力，现阶段每次请求最多返回100条记录，可通过设置日期多次请求获取更多数据。 | ts_code, ann_date, start_date, end_date |
| `disclosure_date` | 获取财报披露计划日期 | ts_code, end_date, pre_date, ann_date |
| `balancesheet` | 获取上市公司资产负债表 | ts_code, ann_date, start_date, end_date |

## 接口详细参数

### `express` 详细说明

**调用说明**: 需要至少2000积分。当前接口只能按单只股票获取其历史数据，如需获取某一季度全部上市公司数据请使用express_vip接口（需5000积分）。

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| ann_date | str | N | 公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期(每个季度最后一天的日期，如20171231表示年报) |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| revenue | float | 营业收入(元) |

| diluted_roe | float | 净资产收益率(摊薄)(%) |
| yoy_net_profit | float | 去年同期修正后净利润 |
| bps | float | 每股净资产 |

### `fina_mainbz` 详细说明

**调用说明**: 需要至少2000积分。单次最大提取100行，总量不限制，可循环获取。如需获取某一季度全部上市公司数据请使用fina_mainbz_vip接口（需5000积分）。
