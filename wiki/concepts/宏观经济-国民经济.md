# 宏观经济-国民经济

> 覆盖GDP、工业增加值、固定资产投资等国民经济核心指标数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `cn_gdp` | 获取国民经济之GDP数据 | q, start_q, end_q, fields |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 cn_gdp 接口
df = pro.cn_gdp()
print(df.head())
```
