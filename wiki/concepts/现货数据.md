# 现货数据

> 提供现货商品的行情和基础数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `sge_basic` | 获取上海黄金交易所现货合约基础信息 | ts_code |
| `sge_daily` | 获取上海黄金交易所现货合约日线行情 | ts_code, trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 sge_basic 接口
df = pro.sge_basic(ts_code="159919.SZ")
print(df.head())
```
