# 债券市场

> 债券市场涵盖银行间市场、交易所市场和柜台市场三个层次，包括回购交易、现券买卖、大宗交易等多种交易方式。回购利率是短期流动性的核心指标，柜台报价反映个人投资者的债券交易成本，大宗交易则展示机构间的场外大额成交情况。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 债券回购(repo_daily) | 以债券为质押的短期资金融通，利率反映流动性松紧 | `repo_daily`（repo_maturity, weight, amount） |
| 银行间质押式回购 | 银行间市场的质押回购，包括 R001/R007/DR001/DR007 等 | R/DR/OR 系列（.IB 后缀） |
| 交易所回购 | 交易所市场的质押回购，包括 GC001/GC007、R-001/R-007 等 | GC（上交所）/ R-（深交所）系列 |
| 柜台债市场(OTC) | 银行柜台面向个人和企业的债券买卖报价 | `bc_otcqt`（buy_yield, sell_yield） |
| 最优报价 | 各银行柜台报价中的最优买入/卖出收益率 | `bc_bestotcqt` |
| 债券大宗交易 | 沪深交易所债券大宗交易汇总数据 | `bond_blk`（price, vol, amount） |
| 大宗交易明细 | 深交所大宗交易的逐笔明细，含买卖营业部 | `bond_blk_detail`（buy_dp, sell_dp） |

## 详细说明

### 债券回购市场

回购交易是债券市场最重要的短期资金融通工具。`repo_daily` 接口汇总了银行间和交易所两大市场的回购日行情数据：

- **银行间质押式回购**：R001/R007（全市场质押）、DR001/DR007（利率债质押，更反映真实资金面）、OR001/OR007（以利率债为质押的定盘回购）
- **交易所回购**：GC001/GC007（上交所国债逆回购）、R-001/R-007（深交所企业债逆回购）
- 加权价（weight）为当日成交量的加权平均利率，是最核心的参考指标

### 柜台债市场

柜台流通式债券报价（`bc_otcqt`）记录各银行柜台的债券买入/卖出收益率，报价机构包括工商银行、建设银行、招商银行、农业银行等。买入收益率（buy_yield）为投资者买入持有的到期收益率（较低），卖出收益率（sell_yield）为投资者卖出到期的收益率（较高），两者之差即为银行的买卖价差收益。

### 大宗交易

债券大宗交易（`bond_blk`）记录沪深交易所单笔大额债券成交数据，包括成交价（price）、成交量（vol）和成交金额（amount）。`bond_blk_detail` 提供深交所大宗交易的逐笔明细，含买方营业部（buy_dp）和卖方营业部（sell_dp），可用于追踪机构资金动向。

### 数据获取示例

```python
pro = ts.pro_api()

# 获取回购日行情
df = pro.repo_daily(trade_date='20200804')

# 获取柜台债报价
df = pro.bc_otcqt(start_date='20240325', end_date='20240329', ts_code='200013.BC')

# 获取债券大宗交易
df = pro.bond_blk(start_date='20210701', end_date='20210930')
```
