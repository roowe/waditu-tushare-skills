# 国债收益率曲线

> 国债收益率曲线是描述国债收益率与剩余期限之间关系的曲线，是债券定价和利率风险管理的基准。中债国债收益率曲线提供即期收益率曲线（curve_type=1）和到期收益率曲线（curve_type=0）两种类型，期限从隔夜到50年，全面反映利率的期限结构。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 收益率曲线 | 利率（收益率）与期限的对应关系图 | `yc_cb` |
| 即期收益率曲线(curve_type=1) | 零息债券的收益率曲线，用于贴现现金流 | `curve_type='1'` |
| 到期收益率曲线(curve_type=0) | 持有到期的年化收益率曲线，最常用 | `curve_type='0'` |
| 期限结构(curve_term) | 从 0 年到 50 年的各期限节点 | `curve_term`（0~50年） |
| 曲线形状 | 正常（上斜）、平坦、倒挂（下斜） | 通过 yield 与 curve_term 的关系判断 |
| 曲线编码(ts_code) | 1001.CB 代表中债国债收益率曲线 | `ts_code='1001.CB'` |

## 详细说明

### 曲线类型：即期 vs 到期

到期收益率曲线（curve_type=0）反映的是附息国债持有到期的年化收益率，是最常用的收益率曲线。即期收益率曲线（curve_type=1）反映的是零息债券的收益率，是金融工程中贴现定价的基础。两条曲线通常形状相似，但即期曲线在短期段波动更为灵敏。

### 期限结构

curve_term 字段表示期限（单位：年），覆盖从 0（隔夜）到 50 年的整条曲线。关键期限包括 1Y、3Y、5Y、10Y、30Y 等。每个交易日约有 50 个期限点（到期+即期合计约 100 个数据点）。

### 曲线形状与经济含义

收益率曲线的形状反映市场对未来经济和利率走势的预期：正常上斜曲线表示经济稳健扩张；平坦曲线暗示增长放缓；倒挂曲线（短期利率高于长期）往往预示经济衰退风险。10Y-2Y 利差是最受关注的曲线斜率指标。

### 数据获取

```python
pro = ts.pro_api('your token')
# 获取中债国债到期收益率曲线
df = pro.yc_cb(ts_code='1001.CB', curve_type='0', trade_date='20200203')
```
