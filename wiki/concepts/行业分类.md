# 行业分类
> A股市场多种行业分类体系及对应的成分股、行情数据接口。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口/字段 |
|---|---|---|---|
| 申万行业分类 | Shenwan Industry Classification | 国内最权威的行业分类体系；2014版28个一级行业/2021版31个一级行业，三级分类 | `index_classify` |
| 中信行业成分 | CITICS Industry Members | 中信行业三级分类体系及成分股 | `ci_index_member` |
| 中信行业行情 | CITICS Industry Daily | 中信行业指数日行情数据 | `citics_index_daily` |
| 东方财富概念板块 | Eastmoney Concept Index | 东方财富概念板块列表及成分 | `dc_index`、`dc_member` |
| 同花顺概念板块 | THS Concept Index | 同花顺概念板块列表及成分 | `ths_index`、`ths_member` |
| 通达信板块 | TDX Sector | 含概念/行业/风格/地域四大类 | `tdx_index`、`tdx_member` |
| 开盘啦题材 | KPL Concept | 开盘啦题材概念成分 | `kpl_concept_cons` |

## 详细说明

### 申万行业分类

申万行业分类是国内使用最广泛的行业分类标准，采用三级分类体系（一级行业 -> 二级行业 -> 三级行业），通过 `index_classify` 接口获取。

**版本对比**：

| 版本 | 一级行业数 | 二级行业数 | 三级行业数 | 参数值 |
|---|---|---|---|---|
| 2014版 | 28 | 104 | 227 | `src='SW2014'` |
| 2021版 | 31 | 134 | 346 | `src='SW2021'` |

2021版新增一级行业：美容护理、环保、煤炭、石油石化；从"纺织服装"改名"纺织服饰"、"电气设备"改名"电力设备"等。

**接口调用**：
```python
# 获取申万一级行业列表
df = pro.index_classify(level='L1', src='SW2021')

# 获取申万二级行业列表
df = pro.index_classify(level='L2', src='SW2021')

# 获取申万三级行业列表
df = pro.index_classify(level='L3', src='SW2021')

# 返回字段：index_code(指数代码)、industry_name(行业名称)、
# parent_code(父级代码)、level(层级)、industry_code(行业代码)、is_pub(是否发布)
```

### 中信行业成分

中信行业分类采用三级体系（`l1_code`/`l2_code`/`l3_code`），通过 `ci_index_member` 接口可按行业查成分股，也可按股票反查所属行业。

**接口调用**：
```python
# 获取某二级行业的成分股
df = pro.ci_index_member(l2_code='CI005835.CI', fields='l2_code,l1_name,ts_code,name')

# 获取某只股票所属行业
df = pro.ci_index_member(ts_code='000001.SZ')

# 返回字段：l1_code/l1_name(一级)、l2_code/l2_name(二级)、
# l3_code/l3_name(三级)、ts_code/name(股票)、in_date/out_date/is_new
```

### 各分类体系特点

| 分类体系 | 特点 | 适用场景 |
|---|---|---|
| 申万 | 最权威，三级结构稳定，行业指数行情完整 | 行业轮动、行业配置 |
| 中信 | 三级分类细致，支持正/反向查询 | 行业归属查询、成分筛选 |
| 东方财富概念 | 概念板块为主，更新及时 | 题材炒作、短期热点 |
| 同花顺概念 | 覆盖面广，用户群体大 | 题材挖掘、散户情绪 |
| 通达信板块 | 含行业/概念/风格/地域四维 | 多维分类、风格轮动 |
| 开盘啦题材 | 短线题材为主 | 打板、游资跟踪 |
