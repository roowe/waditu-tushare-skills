# 市场微观结构
> 描述股票交易活跃度、流动性、市值规模等市场微观特征的数据指标。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口/字段 |
|---|---|---|---|
| 换手率 | Turnover Rate | 当日成交量占总股本的百分比（%） | `turnover_rate` |
| 自由流通换手率 | Free-float Turnover Rate | 当日成交量占自由流通股本的百分比（%） | `turnover_rate_f` |
| 量比 | Volume Ratio | 今日每分钟均量 / 过去5日每分钟均量，>1表示放量 | `volume_ratio` |
| 涨跌幅 | Price Change Percentage | 基于除权昨收计算的涨跌百分比（%） | `pct_chg` |
| 成交量 | Volume | 成交量，单位：手 | `vol` |
| 成交额 | Amount | 成交金额，单位：千元 | `amount` |
| 总股本 | Total Share | 公司总股本，单位：万股 | `total_share` |
| 流通股本 | Float Share | 可在二级市场流通的股本，单位：万股 | `float_share` |
| 自由流通股 | Free Share | 剔除战略持股等非自由流通部分后的股本，单位：万 | `free_share` |
| 总市值 | Total Market Value | 总股本 * 收盘价，单位：万元 | `total_mv` |
| 流通市值 | Circulating Market Value | 流通股本 * 收盘价，单位：万元 | `circ_mv` |
| 涨跌额 | Price Change | 当日收盘价 - 昨收价 | `change` |

## 详细说明

### 数据来源

以上指标均可通过以下两个接口获取：

1. **`stk_factor_pro`** -- 股票技术面因子专业版接口，一次性返回全部微观结构指标和技术因子
2. **`pro_bar`** -- 通用行情接口，通过 `factors=['tor', 'vr']` 参数可获取换手率(`tor`)和量比(`vr`)

### 关键指标解读

- **换手率 vs 自由流通换手率**：`turnover_rate` 基于总股本，`turnover_rate_f` 基于自由流通股本。对于限售股占比较高的股票，两者差异显著，自由流通换手率更能反映真实交易活跃度。

- **量比**：
  - 量比 > 1：今日成交比过去5日活跃（放量）
  - 量比 < 1：今日成交比过去5日清淡（缩量）
  - 量比 > 2：明显放量

- **股本层级关系**：`total_share` >= `float_share` >= `free_share`。自由流通股剔除了大股东持股、战略锁定等非自由交易部分。

- **市值层级关系**：`total_mv` >= `circ_mv`。流通市值 = 流通股本 * 收盘价。

### 接口调用

```python
# stk_factor_pro 获取完整微观结构数据
df = pro.stk_factor_pro(ts_code='000001.SZ', trade_date='20240101')
# 输出包含 turnover_rate, turnover_rate_f, volume_ratio, pct_chg, vol, amount
# 以及 total_share, float_share, free_share, total_mv, circ_mv

# pro_bar 获取换手率和量比
df = ts.pro_bar(ts_code='000001.SZ', start_date='20240101', end_date='20240301', factors=['tor', 'vr'])
# 返回字段 tor(换手率) 和 vr(量比)
```
