# 债券专题

> 覆盖可转债（基础信息/发行/行情/转股/赎回/票面利率）、债券回购日行情、国债收益率曲线、大宗交易、柜台报价等债券市场数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `repo_daily` | 债券回购日行情 | ts_code, trade_date, start_date, end_date |
| `eco_cal` | 获取全球财经日历、包括经济事件数据更新 | date, start_date, end_date, currency |
| `cb_issue` | 获取可转债发行数据 | ts_code, ann_date, start_date, end_date |
| `cb_basic` | 获取可转债基本信息 | ts_code, list_date, exchange |
| `cb_factor_pro` | 获取可转债每日技术面因子数据，用于跟踪可转债当前走势情况，数据由Tushare社区自产，覆盖全历史；输出参数_bfq表示不复权，_qfq表示前复权 _hfq表示后复权，描述中说明了因子的默认传参，如需要特殊参数或者更多因子可以联系管理员评估 | ts_code, start_date, end_date, trade_date |
| `cb_rate` | 获取可转债票面利率 | ts_code |
| `cb_daily` | 获取可转债行情 | ts_code, trade_date, start_date, end_date |
| `cb_call` | 获取可转债到期赎回、强制赎回等信息。数据来源于公开披露渠道，供个人和机构研究使用，请不要用于数据商业目的。 | ts_code, ann_date, start_date, end_date |
| `cb_price_chg` | 获取可转债转股价变动 | ts_code |
| `cb_share` | 获取可转债转股结果 | ts_code, ann_date, start_date, end_date |
| `yc_cb` | 获取中债收益率曲线，目前可获取中债国债收益率曲线即期和到期收益率曲线数据 | ts_code, curve_type, trade_date, start_date |
| `bond_blk` | 获取沪深交易所债券大宗交易数据 | ts_code, trade_date, start_date, end_date |
| `bond_blk_detail` | 获取沪深交易所债券大宗交易数据 | ts_code, trade_date, start_date, end_date |
| `bc_otcqt` | 柜台流通式债券报价 | trade_date, start_date, end_date, ts_code |
| `bc_bestotcqt` | 柜台流通式债券最优报价 | trade_date, start_date, end_date, ts_code |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 repo_daily 接口
df = pro.repo_daily(ts_code="159919.SZ", trade_date="20240101", start_date="20240101")
print(df.head())
```
