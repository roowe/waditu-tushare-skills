# 股票特色数据

> 覆盖龙虎榜、大宗交易、股东人数、质押、增持减持等特色专题数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `stk_ah_comparison` | AH股比价数据，可根据交易日期获取历史 | hk_code, ts_code, trade_date, start_date |
| `ccass_hold_detail` | 获取中央结算系统机构席位持股明细，数据覆盖**全历史**，根据交易所披露时间，当日数据在下一交易日早上9点前完成 | ts_code, hk_code, trade_date, start_date |
| `ccass_hold` | 获取中央结算系统持股汇总数据，覆盖全部历史数据，根据交易所披露时间，当日数据在下一交易日早上9点前完成入库 | ts_code, hk_code, trade_date, start_date |
| `broker_recommend` | 获取券商月度金股，一般1日~3日内更新当月数据 | month |
| `report_rc` | 获取券商（卖方）每天研报的盈利预测数据，数据从2010年开始，每晚19~22点更新当日数据 | ts_code, report_date, start_date, end_date |
| `stk_surv` | 获取上市公司机构调研记录数据 | ts_code, trade_date, start_date, end_date |
| `cyq_chips` | 获取A股每日的筹码分布情况，提供各价位占比，数据从2018年开始，每天18~19点之间更新当日数据 | ts_code, trade_date, start_date, end_date |
| `cyq_perf` | 获取A股每日筹码平均成本和胜率情况，每天18~19点左右更新，数据从2018年开始 | ts_code, trade_date, start_date, end_date |
| `hk_hold` | 获取沪深港股通持股明细，数据来源港交所。 | code, ts_code, trade_date, start_date |
| `stk_nineturn` | 神奇九转（TD序列），识别股价连续9天特定走势判断反转点，数据从20230101开始，每天21点更新 | ts_code, trade_date, freq, start_date |
| `stk_auction_o` | 股票开盘9:30集合竞价数据，每天盘后更新 | ts_code, trade_date, start_date, end_date |
| `stk_factor_pro` | 获取股票每日技术面因子数据，用于跟踪股票当前走势情况，数据由Tushare社区自产，覆盖全历史；输出参数_bfq表示不复权，_qfq表示前复权 _hfq表示后复权，描述中说明了因子的默认传参，如需要特殊参数或者更多因子可以联系管理员评估 | ts_code, trade_date, start_date, end_date |
| `stk_auction_c` | 股票收盘15:00集合竞价数据，每天盘后更新 | ts_code, trade_date, start_date, end_date |

## 接口详细参数

### `stk_ah_comparison` 详细说明

**调用说明**: 积分>=5000 / 单次最大1000行 / 可循环提取
**数据说明**: 每天盘后17:00更新，数据从20250812开始，由于历史不好补充，只能累积

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| hk_code | str | N | 港股股票代码（xxxxx.HK） |
| ts_code | str | N | A股票代码（xxxxxx.SH/SZ/BJ） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| hk_code | str | 港股股票代码 |
| ts_code | str | A股股票代码 |
| trade_date | str | 交易日期 |
| hk_name | str | 港股股票名称 |
| hk_pct_chg | float | 港股股票涨跌幅 |
| hk_close | float | 港股股票收盘价 |
| name | str | A股股票名称 |
| close | float | A股股票收盘价 |
| pct_chg | float | A股股票涨跌幅 |
| ah_comparison | float | 比价（A/H） |
| ah_premium | float | 溢价（A/H）（%） |

### `ccass_hold_detail` 详细说明

**调用说明**: 积分>=8000 / 每分钟300次 / 单次最大6000条 / 可循环或分页提取
**数据说明**: 中央结算系统机构席位持股明细，覆盖全历史，当日数据在下一交易日早上9点前完成入库

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码（如 605009.SH） |
| hk_code | str | N | 港交所代码（如 95009） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代号 |
| name | str | 股票名称 |
| col_participant_id | str | 参与者编号 |
| col_participant_name | str | 机构名称 |
| col_shareholding | str | 持股量（股） |
| col_shareholding_percent | str | 占已发行股份/权证/单位百分比（%） |

### `ccass_hold` 详细说明

**调用说明**: 120积分可试用，5000积分每分钟300次，8000积分以上每分钟500次 / 单次最大5000条
**数据说明**: 中央结算系统持股汇总，覆盖全部历史，当日数据在下一交易日早上9点前完成入库

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码（如 605009.SH） |
| hk_code | str | N | 港交所代码（如 95009） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代号 |
| name | str | 股票名称 |
| shareholding | str | 于中央结算系统的持股量（股） |
| hold_nums | str | 参与者数目（个） |
| hold_ratio | str | 占于上交所上市及交易的A股总数的百分比（%） |
