# 公募基金

> 覆盖公募基金的基础信息、净值、份额、持仓、基金经理等核心数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `fund_nav` | 获取公募基金净值数据 | ts_code, nav_date, market, start_date |
| `fund_div` | 获取公募基金分红数据 | ann_date, ex_date, pay_date, ts_code |
| `fund_basic` | 获取公募基金数据列表，包括场内和场外基金 | ts_code, market, status |
| `fund_factor_pro` | 获取场内基金每日技术面因子数据，用于跟踪场内基金当前走势情况，数据由Tushare社区自产，覆盖全历史；输出参数_bfq表示不复权，描述中说明了因子的默认传参，如需要特殊参数或者更多因子可以联系管理员评估 | ts_code, start_date, end_date, trade_date |
| `fund_portfolio` | 获取公募基金持仓数据，季度更新 | ts_code, symbol, ann_date, period |
| `fund_company` | 获取公募基金管理人列表 | - |
| `fund_manager` | 获取公募基金经理数据，包括基金经理简历等数据 | ts_code, ann_date, name, offset |
| `fund_share` | 获取基金规模数据，包含上海和深圳ETF基金 | ts_code, trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 fund_nav 接口
df = pro.fund_nav(ts_code="159919.SZ", start_date="20240101", end_date="20240131")
print(df.head())
```
