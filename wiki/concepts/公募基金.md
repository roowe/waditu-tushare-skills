# 公募基金

> 覆盖公募基金全生命周期数据：基金分类、净值、分红、持仓、管理团队与规模变动。共 8 个接口。

## 核心概念

### 基金分类

通过 `fund_basic` 获取全量基金列表，区分场内基金（ETF/LOF，market='E'）和场外基金（market='O'）。关键字段：
- **fund_type**：投资类型（股票型/混合型/债券型/货币型等）
- **invest_type**：投资风格（成长型/价值型/平衡型等）
- **m_fee / c_fee**：管理费 / 托管费
- **status**：存续状态（D摘牌 / I发行 / L已上市）

### 基金净值

`fund_nav` 提供三类净值指标（2000 积分以上）：
- **unit_nav**：单位净值，即每份基金的实际价值
- **accum_nav**：累计净值，含历史分红
- **adj_nav**：复权单位净值，分红再投资后的真实回报

### 基金分红

`fund_div` 记录分红全流程（400 积分以上）。关键时间节点：
- **ann_date** → 公告日
- **record_date** → 权益登记日
- **ex_date** → 除息日
- **pay_date** → 派息日
- **div_cash** → 每股派息金额（元）

### 基金持仓

`fund_portfolio` 季度更新（5000 积分以上高频访问），展示基金持有的股票明细：
- **mkv**：持有股票市值（元）
- **amount**：持有股票数量（股）
- **stk_mkv_ratio**：占股票市值比（%）
- **stk_float_ratio**：占流通股本比例（%）

### 基金管理人

`fund_company` 一次性提取全量基金公司列表（1500 积分），包含公司注册地址、法人、总经理、员工数、注册资本等。

### 基金经理

`fund_manager` 提供基金经理履历数据（500 积分起，2000 积分以上高频）：
- **begin_date / end_date**：任职/离任日期
- **resume**：个人简历
- 支持 ts_code 逗号分隔多基金查询

### 基金规模

`fund_share` 追踪基金份额变动（2000 积分以上），以万份为单位。支持多基金同时提取（逗号分隔）。

## 接口一览

| 接口名 | 说明 | 关键参数 | 积分 |
|--------|------|----------|------|
| `fund_basic` | 基金列表（场内+场外） | market, status, ts_code | 2000 |
| `fund_nav` | 基金净值（单位/累计/复权） | ts_code, nav_date, market | 2000 |
| `fund_div` | 基金分红 | ann_date, ex_date, pay_date, ts_code | 400 |
| `fund_portfolio` | 基金季度持仓 | ts_code, symbol, ann_date, period | 5000 |
| `fund_company` | 基金管理人列表 | 无 | 1500 |
| `fund_manager` | 基金经理简历 | ts_code, ann_date, name | 500 |
| `fund_share` | 基金份额规模 | ts_code, trade_date, market | 2000 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 场内基金列表
df = pro.fund_basic(market='E')

# 某基金净值走势
df = pro.fund_nav(ts_code='165509.SZ', start_date='20240101', end_date='20240131')

# 基金季度持仓
df = pro.fund_portfolio(ts_code='001753.OF')

# 基金经理履历
df = pro.fund_manager(ts_code='150018.SZ')

# 多基金份额变动
df = pro.fund_share(ts_code='150018.SZ,150008.SZ')
```
