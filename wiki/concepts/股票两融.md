# 股票两融

> 覆盖融资融券余额、明细、转融通等两融数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `slb_len_mm` | 做市借券交易汇总 | trade_date, ts_code, start_date, end_date |
| `margin_detail` | 获取沪深两市每日融资融券明细 | trade_date, ts_code, start_date, end_date |
| `margin` | 获取融资融券每日交易汇总数据 | trade_date, start_date, end_date, exchange_id |
| `margin_secs` | 获取沪深京三大交易所融资融券标的（包括ETF），每天盘前更新 | ts_code, trade_date, exchange, start_date |
| `slb_sec_detail` | 转融券交易明细 | trade_date, ts_code, start_date, end_date |
| `slb_sec` | 转融通转融券交易汇总 | trade_date, ts_code, start_date, end_date |
| `slb_len` | 转融通融资汇总 | trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 slb_len_mm 接口
df = pro.slb_len_mm(trade_date="20240101", ts_code="159919.SZ", start_date="20240101")
print(df.head())
```
