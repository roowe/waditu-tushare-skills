# 融资融券
> 融资融券是A股市场重要的杠杆交易机制，融资为做多提供杠杆，融券为做空提供工具，两融余额变化是衡量市场杠杆水平和投资者情绪的重要指标。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 融资买入 | Margin Buy (rzmre) | 投资者向券商借入资金买入证券，属于做多杠杆操作 | `margin` / `margin_detail` |
| 融资偿还 | Margin Repay (rzche) | 投资者卖出证券或以现金偿还融资债务 | `margin` / `margin_detail` |
| 融资余额 | Margin Balance (rzye) | 前日融资余额 + 本日融资买入 - 本日融资偿还 | `margin` / `margin_detail` |
| 融券卖出 | Short Sell (rqmcl) | 投资者向券商借入证券并卖出，属于做空操作 | `margin` / `margin_detail` |
| 融券偿还 | Short Repay (rqchl) | 投资者买回证券归还券商或以现券偿还 | `margin_detail` |
| 融券余量 | Short Volume (rqyl) | 前日融券余量 + 本日融券卖出 - 本日融券买入 - 现券偿还 | `margin` / `margin_detail` |
| 融券余额 | Short Balance (rqye) | 融券余量 x 当日收盘价 | `margin` / `margin_detail` |
| 标的证券 | Underlying Security | 可进行融资融券交易的证券名单，由交易所定期调整 | `margin_secs` |
| 转融通 | Securities Relending | 证券金融公司将自有或借入的证券/资金转借给券商，再由券商提供给客户 | `slb_sec` |
| 转融资 | Securities Relending-Fund | 证券金融公司将资金转借给券商用于融资业务 | `slb_len` |
| 做市借券 | Market Making Borrow | 为做市商提供券源支持，提升市场流动性 | `slb_len_mm` |

## 详细说明

### 融资融券基本原理

融资融券（两融）是投资者向券商借入资金买入证券（融资）或借入证券卖出（融券）的交易方式。两融余额反映了市场的杠杆水平：

- **融资余额上升**：投资者看好后市，愿意借钱买股，市场情绪偏多
- **融资余额下降**：投资者看空或降杠杆，卖出偿还融资，市场情绪偏空
- **融券余额上升**：投资者看空后市，借券卖出，预期价格下跌
- **融券余额下降**：空头回补，买券偿还，看空情绪减弱

### 余额计算公式

```
融资余额 = 前日融资余额 + 本日融资买入额 - 本日融资偿还额
融券余量 = 前日融券余量 + 本日融券卖出量 - 本日融券买入量 - 本日现券偿还量
融券余额 = 融券余量 x 当日收盘价
融资融券余额 = 融资余额 + 融券余额
```

### 数据层次

**交易所汇总（margin）**
- 按交易所（上交所SSE、深交所SZSE、北交所BSE）统计的两融汇总数据
- 包含融资余额、融资买入额、融资偿还额、融券余额、融券卖出量等

**个股明细（margin_detail）**
- 按个股维度展示每日融资融券明细数据
- 2019年9月10日后包含股票名称字段
- 单次最大返回6000行

**标的证券（margin_secs）**
- 沪深京三大交易所融资融券标的名单（含ETF）
- 每天盘前更新，交易所会定期调整标的范围

### 转融通机制

转融通是两融业务的上游机制，由证券金融公司（证金公司）作为中介：

- **转融券（slb_sec）**：证金公司将证券转借给券商，券商再提供给客户融券卖出。数据包含期初余量、融出数量、期末余量和期末余额
- **转融资（slb_len）**：证金公司将资金转借给券商用于融资业务。数据包含期初余额、竞价成交金额、再借成交金额、偿还金额和期末余额

转融通扩大了两融业务的券源和资金来源，为市场提供更丰富的交易工具。
