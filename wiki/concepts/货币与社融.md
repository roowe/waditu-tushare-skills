# 货币与社融

> 货币供应量（M0/M1/M2）和社会融资规模是衡量金融体系流动性和实体经济融资状况的两大核心指标。M1-M2 剪刀差反映资金活化程度，社融增量反映实体经济从金融体系获得的资金总量。两者共同构成宏观金融分析的基础框架。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| M0(流通现金) | 流通中的现金 | `cn_m`（m0, m0_yoy） |
| M1(窄货币) | M0 + 活期存款，反映即时购买力 | `cn_m`（m1, m1_yoy） |
| M2(广义货币) | M1 + 定期存款，反映总体货币供给 | `cn_m`（m2, m2_yoy） |
| M1-M2 剪刀差 | M1 增速与 M2 增速之差，正值表示资金活化 | m1_yoy - m2_yoy |
| 社会融资规模(当月增量) | 实体经济当月从金融体系获得的新增资金 | `sf_month`（inc_month） |
| 社会融资规模(累计增量) | 年初至当月的社融累计增量 | `sf_month`（inc_cumval） |
| 社会融资规模(存量) | 期末社融余额总量 | `sf_month`（stk_endval） |
| 社融构成 | 银行贷款/信托/委托/未贴现汇票/债券/股票/政府债券等 | 各分项数据 |

## 详细说明

### 货币供应量

`cn_m` 接口提供月度货币供应量数据，600 积分即可调用。三个层次的货币供应量定义如下：

- **M0**：流通中的现金，是最狭义的货币度量
- **M1**（窄货币）：M0 + 单位活期存款，反映经济中的即时购买力和交易活跃度
- **M2**（广义货币）：M1 + 单位定期存款 + 个人储蓄存款 + 其他存款，是衡量货币供给总量最常用的指标

每个层次均有当月值、同比增速（yoy）和环比增速（mom）三个维度。

```python
pro = ts.pro_api()
df = pro.cn_m(start_m='201901', end_m='202003', fields='month,m0,m1,m2,m1_yoy,m2_yoy')
```

### M1-M2 剪刀差

M1 增速与 M2 增速的差值是判断经济活力的重要指标。当 M1 增速 > M2 增速（正剪刀差），说明活期存款增速快于定期存款，企业和居民交易活跃，经济向好；当 M1 增速 < M2 增速（负剪刀差），说明资金从活期转向定期，经济活力下降。持续扩大的负剪刀差通常预示经济下行压力。

### 社会融资规模

`sf_month` 接口提供月度社会融资规模数据（需 2000 积分），核心字段包括：

- **inc_month**：社融当月增量（亿元），是最受市场关注的月度金融数据
- **inc_cumval**：社融年初至今累计增量（亿元）
- **stk_endval**：社融存量期末值（万亿元）

社融是衡量实体经济从金融体系获得资金总量的综合指标，其构成包括：
- 表内融资：人民币贷款、外币贷款
- 表外融资：信托贷款、委托贷款、未贴现银行承兑汇票
- 直接融资：企业债券、非金融企业境内股票
- 其他：政府债券、投资性房地产、保险公司赔偿等

### 社融与货币供应量的关系

社融是从"需求端"衡量实体经济获得的资金，M2 是从"供给端"衡量金融体系的负债。社融增速高于 M2 增速说明实体融资需求旺盛，反之说明资金在金融体系内空转。两者结合 [[利率体系]] 分析，可以更全面地判断货币政策的松紧程度。
