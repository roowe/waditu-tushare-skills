> 大宗交易是A股市场中大额股份的协议转让机制，在正常交易时段之外通过大宗交易系统完成，为机构投资者的巨额交易提供便利，同时也是股东减持的重要渠道。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|-------------|
| 大宗交易 | 单笔交易数量或金额达到规定门槛的证券交易，通过大宗交易系统申报成交 | `block_trade` |
| 准入门槛 | A股: 单笔>=30万股或>=200万元；B股: 单笔>=3万股或>=20万港币/美元 | - |
| 交易时间 | 正常交易日的15:00-15:30（沪深）/ 9:15-11:30/13:00-15:30（北交所） | - |
| 折价率 | 大宗交易成交价相对当日收盘价的折扣比例，通常折价5%-15% | block_trade(price) vs daily(close) |
| 协议大宗交易 | 买卖双方事先协商好价格和数量后通过大宗交易系统成交 | - |
| 盘后定价大宗交易 | 买卖双方不需事先协商，通过盘后定价大宗交易方式成交 | - |
| 减持通道 | 大股东/董监高通过大宗交易减持，受减持新规约束 | - |
| 买方营业部 | 大宗交易买方所在的券商营业部 | block_trade(buyer) |
| 卖方营业部 | 大宗交易卖方所在的券商营业部 | block_trade(seller) |
| 倒手交易 | 同一股票在同一营业部或关联营业部之间的大宗交易，可能暗示利益输送 | - |

## 详细说明

### 大宗交易规则

1. **准入门槛**:
   - A股: 单笔申报数量>=30万股 或 成交金额>=200万元
   - B股: 单笔申报数量>=3万股 或 成交金额>=20万港币/美元
   - 基金: 单笔申报数量>=200万份 或 成交金额>=200万元
   - 债券: 单笔申报数量>=5000张 或 成交金额>=50万元

2. **交易时间**:
   - 沪深交易所: 15:00-15:30（接受申报），成交确认后即时公布
   - 北交所: 9:15-11:30、13:00-15:30

3. **价格限制**:
   - 协议大宗交易: 成交价在该证券当日涨跌幅价格范围内
   - 有价格涨跌幅限制的证券: 成交价由买卖双方在当日价格涨跌幅限制范围内协商确定
   - 无价格涨跌幅限制的证券: 成交价由买卖双方自行协商确定

### 折价率分析

- **正常折价(5%-15%)**: 大宗交易因流动性成本通常折价成交
- **高折价(>15%)**: 可能暗示急于脱手或利益输送
- **平价或溢价**: 买方看好，或为战略投资者入场
- **计算**: 折价率 = (大宗交易价 - 当日收盘价) / 当日收盘价

### 大宗交易与减持的关系

- 减持新规(2017年修订): 大股东通过大宗交易减持的，受让方在6个月内不得转让
- 这导致大宗交易受让方成为"过桥资金"的通道
- 卖方营业部+买方营业部的组合分析可揭示减持路径

### 量化分析应用

- 大额折价大宗交易可能预示股东看空
- 溢价大宗交易可能预示战略投资者入场
- 连续多日大宗交易可能预示持续减持
- 买方营业部集中度分析：知名机构营业部的大宗交易更值得关注

## 相关接口

**block_trade** -- 大宗交易
- 积分要求: 300积分
- 单次最大1000条

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| price | float | 成交价 |
| vol | float | 成交量(万股) |
| amount | float | 成交金额 |
| buyer | str | 买方营业部 |
| seller | str | 卖方营业部 |
