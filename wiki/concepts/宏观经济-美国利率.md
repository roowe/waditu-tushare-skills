# 宏观经济-美国利率

> 覆盖美联储利率、美国国债收益率等国际利率数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `us_trycr` | 国债实际收益率曲线利率 | date, start_date, end_date, fields |
| `us_tycr` | 获取美国每日国债收益率曲线利率 | date, start_date, end_date, fields |
| `us_tltr` | 国债长期利率 | date, start_date, end_date, fields |
| `us_trltr` | 国债实际长期利率平均值 | date, start_date, end_date, fields |
| `us_tbr` | 获取美国短期国债利率数据 | date, start_date, end_date, fields |

## 接口详细参数

### `us_trycr` 详细说明

**调用说明**: 需120积分；单次最大2000行，可循环获取
**数据说明**: 日频数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | str | N | 日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| fields | str | N | 指定输出字段 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| date | str | 日期 |
| y5 | float | 5年期 |
| y7 | float | 7年期 |
| y10 | float | 10年期 |
| y20 | float | 20年期 |
| y30 | float | 30年期 |

### `us_tycr` 详细说明

**调用说明**: 需120积分；单次最大2000条，可循环获取
**数据说明**: 日频数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | str | N | 日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| fields | str | N | 指定输出字段（e.g. fields='m1,y1'） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| date | str | 日期 |
| m1 | float | 1月期 |
| m2 | float | 2月期 |
| m3 | float | 3月期 |
| m4 | float | 4月期（数据从20221019开始） |
| m6 | float | 6月期 |
| y1 | float | 1年期 |
| y2 | float | 2年期 |
| y3 | float | 3年期 |
| y5 | float | 5年期 |
| y7 | float | 7年期 |
| y10 | float | 10年期 |
| y20 | float | 20年期 |
| y30 | float | 30年期 |

### `us_tltr` 详细说明

**调用说明**: 需120积分；单次最大2000行，可循环获取
**数据说明**: 日频数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | str | N | 日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| fields | str | N | 指定字段 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| date | str | 日期 |
| ltc | float | 收益率 LT COMPOSITE (>10 Yrs) |
| cmt | float | 20年期CMT利率(TREASURY 20-Yr CMT) |
| e_factor | float | 外推因子EXTRAPOLATION FACTOR |

### `us_trltr` 详细说明

**调用说明**: 需120积分；单次最大2000行，可循环获取
