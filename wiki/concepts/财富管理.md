# 财富管理

> 覆盖基金销售行业数据，包括基金销售机构、渠道等财富管理相关数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `fund_sales_ratio` | 获取各渠道公募基金销售保有规模占比数据，年度更新 | 年份 |
| `fund_sales_vol` | 获取销售机构公募基金销售保有规模数据，本数据从2021年Q1开始公布，季度更新 | year, quarter, name |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 fund_sales_ratio 接口
df = pro.fund_sales_ratio()
print(df.head())
```
