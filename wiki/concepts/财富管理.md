# 财富管理

> 覆盖公募基金销售行业的渠道分布和机构排名数据，适用于财富管理行业研究和渠道竞争分析。共 2 个接口。

## 核心概念

### 基金销售渠道

国内公募基金销售渠道分为五大类：

| 渠道 | 说明 | 典型代表 |
|------|------|----------|
| 商业银行 | 银行代销渠道 | 招商银行、工商银行、建设银行 |
| 证券公司 | 券商代销渠道 | 华泰证券、中信证券 |
| 基金公司直销 | 基金公司自有渠道 | 各基金公司官网/APP |
| 独立销售机构 | 第三方互联网平台 | 天天基金、蚂蚁基金 |
| 其他 | 期货公司、投资咨询等 | - |

### 各渠道保有规模占比

`fund_sales_ratio` 提供年度更新的各渠道公募基金销售保有规模占比（2015年起），字段包括：
- **bank**：商业银行占比（%）
- **sec_comp**：证券公司占比（%）
- **fund_comp**：基金公司直销占比（%）
- **indep_comp**：独立基金销售机构占比（%）
- **rests**：其他占比（%）

趋势观察：基金公司直销占比从 2015 年的 61.9% 逐年下降，独立销售机构从 2.14% 快速增长至 2019 年的 11.03%。

### 销售机构保有规模排名

`fund_sales_vol` 提供季度更新的机构级保有规模排名（2021年Q1起），字段包括：
- **fund_scale**：股票+混合公募基金保有规模（亿元）
- **scale**：非货币市场公募基金保有规模（亿元）
- **rank**：排名

行业集中度较高：头部机构（招商银行、蚂蚁基金、工商银行等）占据主要市场份额。

## 接口一览

| 接口名 | 说明 | 关键参数 | 更新频率 |
|--------|------|----------|----------|
| `fund_sales_ratio` | 各渠道保有规模占比 | year | 年度 |
| `fund_sales_vol` | 销售机构保有规模排名 | year, quarter, name | 季度 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 各渠道保有规模占比历史
df = pro.fund_sales_ratio()
# 输出: year, bank, sec_comp, fund_comp, indep_comp, rests

# 2021年Q1机构保有规模排名
df = pro.fund_sales_vol()

# 查询某机构历史排名
df = pro.fund_sales_vol(name='招商银行股份有限公司')
```
