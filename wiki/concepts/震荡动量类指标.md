# 震荡动量类指标
> 衡量价格超买超卖状态、动量强弱及市场情绪的技术分析指标集合，数据来源于 `stk_factor_pro` 接口。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口/字段 |
|---|---|---|---|
| KDJ | Stochastic Oscillator | K/D/J 值，K=RSV的M1日均线，D=K的M2日均线，J=3K-2D；参数 N=9/M1=3/M2=3 | `kdj_k_*`、`kdj_d_*`、`kdj_*`(J值) |
| RSI | Relative Strength Index | 相对强弱指标，N日内上涨均值占比；周期 6/12/24 日 | `rsi_*_6`、`rsi_*_12`、`rsi_*_24` |
| W&R | Williams %R | 威廉指标，(HN-C)/(HN-LN)*100；参数 N=10/N1=6 | `wr_*`(N=10)、`wr1_*`(N1=6) |
| CCI | Commodity Channel Index | 顺势指标，价格偏离统计均值程度；参数 N=14 | `cci_*` |
| MTM | Momentum | 动量指标，MTM=C-C(N)，MTMMA=MTM的M日均值；参数 N=12/M=6 | `mtm_*`、`mtmma_*` |
| ROC | Rate of Change | 变动率，ROC=(C-C(N))/C(N)*100，MAROC=ROC的M日均值；参数 N=12/M=6 | `roc_*`、`maroc_*` |
| BIAS | Bias Ratio | 乖离率，(C-MA)/MA*100；三组参数 L1=6/L2=12/L3=24 | `bias1_*`(6日)、`bias2_*`(12日)、`bias3_*`(24日) |
| MFI | Money Flow Index | 成交量版RSI，结合价格与成交量；参数 N=14 | `mfi_*` |
| BRAR | BRAR Sentiment | 情绪指标，AR=开盘价动能，BR=收盘价动能；参数 M1=26 | `brar_ar_*`、`brar_br_*` |
| PSY | Psychological Line | 心理线，N日内上涨天数占比；参数 N=12/M=6 | `psy_*`、`psyma_*` |
| OBV | On Balance Volume | 能量潮，上涨日累加成交量、下跌日累减 | `obv_*` |
| VR | Volume Ratio | 容量比率，上涨日成交量总和/下跌日成交量总和；参数 M1=26 | `vr_*` |
| ASI | Accumulation Swing Index | 振动升降指标，ASI衡量真实市场波动；参数 M1=26/M2=10 | `asi_*`、`asit_*`(ASI的M日均值) |
| EMV | Ease of Movement | 简易波动指标，结合价格变化与成交量；参数 N=14/M=9 | `emv_*`、`maemv_*` |
| 神奇九转 | TD Sequential | TD序列，`stk_nineturn` 接口提供 up_count/down_count 计数 | `stk_nineturn` |
| ATR | Average True Range | 真实波幅N日均值，衡量波动性；参数 N=20 | `atr_*` |
| BBI | Bull and Bear Index | 多空指标，多周期MA均值；参数 M1=3/M2=6/M3=12/M4=20 | `bbi_*` |
| MASS | MASS Index | 梅斯线，波动率压缩/扩张识别；参数 N1=9/N2=25/M=6 | `mass_*`、`ma_mass_*` |
| DPO | Detrended Price Oscillator | 区间震荡线，消除趋势影响的价格震荡；参数 M1=20/M2=10/M3=6 | `dpo_*`、`madpo_*` |
| EXPMA | Exponential Moving Average | 指数平均数指标，双线交叉系统；参数 N1=12/N2=50 | `expma_12_*`、`expma_50_*` |
| CR | CR Momentum | CR价格动量指标；参数 N=20 | `cr_*` |

## 详细说明

所有震荡动量类指标均通过 `stk_factor_pro` 接口获取，每个指标字段名后缀为复权类型：
- `_bfq` -- 不复权
- `_hfq` -- 后复权
- `_qfq` -- 前复权

**典型用法**：
- **KDJ/RSI**：超买超卖判断（KDJ的J>100或RSI>80为超买区）
- **CCI**：+100以上为强势，-100以下为弱势
- **BIAS**：乖离率过大预示均值回归
- **OBV/VR**：量价背离预警
- **ATR**：止损位设置与波动率评估
- **神奇九转**：计数到达9时可能出现趋势反转

**接口调用**：
```python
# 获取单只股票全部技术因子（含震荡动量指标）
df = pro.stk_factor_pro(ts_code='000001.SZ', start_date='20240101', end_date='20240301')
# 神奇九转单独接口
df = pro.stk_nineturn(ts_code='000001.SZ', trade_date='20240101')
```
