# 大模型语料专题数据

> 面向大模型训练和语料构建的专题数据，包括金融文本、公告摘要等结构化语料

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `anns_d` | 获取全量公告数据，提供pdf下载URL | ts_code, ann_date, start_date, end_date |
| `irm_qa_sh` | 获取上交所e互动董秘问答文本数据。上证e互动是由上海证券交易所建立、上海证券市场所有参与主体无偿使用的沟通平台,旨在引导和促进上市公司、投资者等各市场参与主体之间的信息沟通,构建集中、便捷的互动渠道。本接口数据记录了以上沟通问答的文本数据。 | ts_code, trade_date, start_date, end_date |
| `research_report` | 获取券商研究报告-个股、行业等，历史数据从20170101开始提供，增量每天两次更新 | trade_date, start_date, end_date, report_type |
| `npr` | 获取国家行政机关公开披露的各类法规、条例政策、批复、通知等文本数据。 | org, start_date, end_date, ptype |
| `news` | 获取主流新闻网站的快讯新闻数据,提供超过6年以上历史新闻。 | start_date, end_date, src |
| `cctv_news` | 获取新闻联播文字稿数据，数据开始于2017年。 | date |
| `major_news` | 获取长篇通讯信息，覆盖主要新闻资讯网站，提供超过8年历史新闻。 | src, start_date, end_date |
| `irm_qa_sz` | 互动易是由深交所官方推出,供投资者与上市公司直接沟通的平台,一站式公司资讯汇集,提供第一手的互动问答、投资者关系信息、公司声音等内容。 | ts_code, trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 anns_d 接口
df = pro.anns_d(ts_code="159919.SZ", ann_date="20240101", start_date="20240101")
print(df.head())
```
