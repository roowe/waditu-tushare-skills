# 可转债

> 可转债（Convertible Bond, CB）是上市公司发行的可转换为正股股票的债券，兼具债性与股性；可交换债（Exchangeable Bond, EB）则是股东以其持有的股票为标的发行的交换债券。两者均以正股（stk_code）价格为核心驱动，通过转股价（conv_price）在债券与股票之间切换价值锚定。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 可转债(CB)/可交换债(EB) | 转债类型，CB 由上市公司发行，EB 由股东发行 | `cb_type`（cb_basic） |
| 正股(stk_code) | 可转债对应的标的股票 | `stk_code`, `stk_short_name` |
| 转股价(conv_price)/转股期 | 将债券转换为股票的价格，以及可转股的时间区间 | `conv_price`, `conv_start_date`, `conv_end_date` |
| 转股价值(cb_value) | 正股价格 / 转股价 x 100，衡量转债的股性 | `cb_value`（cb_daily） |
| 转股溢价率(cb_over_rate) | 转债价格高于转股价值的比率，反映市场预期 | `cb_over_rate`（cb_daily） |
| 纯债价值(bond_value) | 按剩余现金流贴现计算的债券底价 | `bond_value`（cb_daily） |
| 纯债溢价率(bond_over_rate) | 转债价格高于纯债价值的比率 | `bond_over_rate`（cb_daily） |
| 回售条款(put_clause) | 正股持续低迷时持有人可按约定价格回售给发行人 | `put_clause`（cb_basic） |
| 赎回条款(call_clause) | 正股持续上涨时发行人可强制赎回（强赎） | `call_clause`（cb_basic）; `cb_call` |
| 特别向下修正条款(reset_clause) | 正股大幅下跌时发行人可下调转股价以促成转股 | `reset_clause`（cb_basic）; `cb_price_chg` |
| 票面利率/递进利率 | 可转债通常采用逐年递进的票面利率 | `cb_rate`（coupon_rate, rate_freq） |
| 信用等级/担保 | 发行信用评级及担保方式 | `issue_rating`, `guarantor`, `guarantee_type` |
| 发行(中签率/配售比例) | 网上网下发行的中签率、配售比例等 | `cb_issue`（onl_winning_rate, shd_ration_ratio） |
| 转股结果跟踪 | 每次及累计转股数量、金额、比例 | `cb_share`（convert_vol, acc_convert_ratio） |

## 详细说明

### 基本信息与分类

通过 `cb_basic` 接口获取可转债/可交换债的基础信息，包括转股代码（cb_code）、正股代码（stk_code）、发行期限（maturity）、面值（par）、初始转股价（first_conv_price）以及最新转股价（conv_price）等核心字段。`cb_type` 字段区分可转债（CB）与可交换债（EB）。

### 转股价变动与向下修正

转股价是可转债定价的核心参数。当正股价格持续下跌满足特别向下修正条款时，发行人可下调转股价。通过 `cb_price_chg` 接口可追踪每次转股价变动记录，包括修正前价格（convertprice_bef）和修正后价格（convertprice_aft）。

### 赎回与强赎机制

可转债的赎回分为到期赎回和强制赎回两种类型。当正股价格在特定期间内持续高于转股价一定比例时，发行人有权按约定价格强制赎回未转股的债券。通过 `cb_call` 接口可获取赎回类型（call_type: 到赎/强赎）、是否赎回状态（is_call: 已满足强赎条件/公告提示强赎/公告实施强赎/公告不强赎等）以及赎回价格（call_price）等信息。

### 行情与估值指标

`cb_daily` 接口提供可转债的日线行情数据，除常规 OHLCV 外，还包含纯债价值（bond_value）、纯债溢价率（bond_over_rate）、转股价值（cb_value）和转股溢价率（cb_over_rate）四项关键估值指标。转股溢价率越低，转债价格与正股走势越趋同；纯债价值则为转债价格提供底部支撑。

### 发行与票面利率

`cb_issue` 接口提供网上发行的中签率（onl_winning_rate）、有效申购数量（onl_pch_vol）、老股东配售比例（shd_ration_ratio）等发行数据。票面利率通过 `cb_rate` 接口获取，通常为逐年递进结构，例如 0.5% -> 0.7% -> 1.0% -> 1.5% -> 2.5% -> 3.0%。

### 转股结果跟踪

通过 `cb_share` 接口可获取每次转股的数量（convert_vol）、金额（convert_val）、比例（convert_ratio）以及累计转股比例（acc_convert_ratio）和剩余金额（remain_size），用于跟踪转债的转股进度和市场行为。
