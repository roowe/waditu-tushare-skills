# 通胀指标

> CPI（居民消费价格指数）和 PPI（工业生产者出厂价格指数）是衡量通货膨胀/通货紧缩的两大核心指标。CPI 反映居民消费品和服务价格的变动，PPI 反映工业品出厂价格的变动。两者共同构成价格体系的全景图，是货币政策决策的重要依据。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| CPI(居民消费价格指数) | 衡量居民消费品和服务价格变动的指标 | `cn_cpi`（nt_val, nt_yoy, nt_mom） |
| CPI 同比 | 与去年同月相比的价格变动百分比 | `nt_yoy`（全国同比） |
| CPI 环比 | 与上月相比的价格变动百分比 | `nt_mom`（全国环比） |
| CPI 城市/农村 | 分别统计城市和农村的价格变动 | `town_yoy`, `cnt_yoy` |
| PPI(工业生产者出厂价格指数) | 衡量工业品出厂价格变动的指标 | `cn_ppi`（ppi_yoy, ppi_mom） |
| PPI 生产资料/生活资料 | 按用途分为生产资料和生活资料两大类 | `ppi_mp_yoy`, `ppi_cg_yoy` |
| PPI 采掘/原料/加工 | 生产资料的细分行业 | `ppi_mp_qm_yoy`, `ppi_mp_rm_yoy`, `ppi_mp_p_yoy` |
| 通胀 vs 通缩 | CPI 持续上涨为通胀，持续下跌为通缩 | 综合 nt_yoy 和 ppi_yoy 判断 |

## 详细说明

### CPI 数据结构

`cn_cpi` 接口提供月度 CPI 数据，包括全国（nt_）、城市（town_）、农村（cnt_）三个维度的当月值（val）、同比（yoy）、环比（mom）和累计值（accu）。数据按月度（YYYYMM）组织，600 积分即可调用。

```python
pro = ts.pro_api()
# 获取 CPI 数据
df = pro.cn_cpi(start_m='201801', end_m='201903', fields='month,nt_val,nt_yoy,nt_mom')
```

### PPI 数据结构

`cn_ppi` 接口提供月度 PPI 数据，按行业维度细分为：
- **全部工业品**（ppi_yoy/mom/accu）
- **生产资料**：采掘业（ppi_mp_qm_）、原料业（ppi_mp_rm_）、加工业（ppi_mp_p_）
- **生活资料**：食品类（ppi_cg_f_）、衣着类（ppi_cg_c_）、一般日用品（ppi_cg_adu_）、耐用消费品（ppi_cg_dcg_）

每个细分项均有同比（yoy）、环比（mom）、累计同比（accu）三个维度。

### CPI 与 PPI 的关系

CPI 和 PPI 分别从消费端和生产端反映价格变动。PPI 向 CPI 的传导是通胀分析的核心：当 PPI 持续上涨但 CPI 未同步上行时，说明上游涨价压力未能传导至下游，企业利润受挤压；当两者同步上行时，说明通胀压力已全面扩散。PPI-CPI 剪刀差是判断经济周期位置的重要指标。

### 通胀与通缩的判断

一般而言，CPI 同比持续高于 3% 被视为通胀预警信号，持续低于 1% 或为负值则面临通缩风险。PPI 同比为负通常意味着工业需求疲弱。两者的综合走势对货币政策和 [[利率体系]] 有直接影响。
