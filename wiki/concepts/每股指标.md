# 每股指标
> 将公司财务数据按每股口径标准化，便于不同规模公司之间的横向比较。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 基本每股收益 | Basic EPS | 净利润 / 期末总股本 | `fina_indicator.eps` / `income.basic_eps` |
| 稀释每股收益 | Diluted EPS | 考虑潜在稀释效应后的每股收益 | `fina_indicator.dt_eps` / `income.diluted_eps` |
| 期末摊薄每股收益 | Diluted2 EPS | 按期末股本全面摊薄 | `fina_indicator.diluted2_eps` |
| 扣非每股收益 | Deducted EPS | 扣除非经常损益后的每股收益 | `fina_indicator.dt_eps`（扣非口径） |
| 每股净资产 | BPS | 股东权益 / 总股本 | `fina_indicator.bps` |
| 每股经营现金流 | OCFPS | 经营现金流净额 / 总股本 | `fina_indicator.ocfps` |
| 每股现金流量 | CFPS | 现金流量净额 / 总股本 | `fina_indicator.cfps` |
| 每股资本公积 | Capital Reserve PS | 资本公积 / 总股本 | `fina_indicator.capital_rese_ps` |
| 每股盈余公积 | Surplus Reserve PS | 盈余公积 / 总股本 | `fina_indicator.surplus_rese_ps` |
| 每股未分配利润 | Undistributed Profit PS | 未分配利润 / 总股本 | `fina_indicator.undist_profit_ps` |
| 每股营业收入 | Revenue PS | 营业收入 / 总股本 | `fina_indicator.revenue_ps` |
| 每股营业总收入 | Total Revenue PS | 营业总收入 / 总股本 | `fina_indicator.total_revenue_ps` |
| 每股留存收益 | Retained PS | 留存收益 / 总股本 | `fina_indicator.retainedps` |

## 详细说明

### EPS 体系
EPS 是最基础的每股指标，直接反映每股盈利水平：
- `basic_eps` / `eps`：基本每股收益，来自 `income` 和 `fina_indicator` 两个接口。
- `diluted_eps` / `dt_eps`：稀释每股收益，考虑期权、可转债等潜在股本增加的影响。
- `diluted2_eps`：期末摊薄口径，使用期末总股本全面摊薄。

### 每股净资产（BPS）
BPS 是 PB 估值的基础：PB = 股价 / BPS。当股价低于 BPS 时称为"破净"。

### 每股现金流
- `ocfps`：每股经营活动现金流，反映主业造血能力的每股水平。
- `cfps`：每股全部现金流量净额。
- 当 EPS 远高于 OCFPS 时，可能存在利润质量疑问。

### 每股权益构成
- `capital_rese_ps`：每股资本公积，主要来自股本溢价。
- `surplus_rese_ps`：每股盈余公积，法定提取的留存。
- `undist_profit_ps`：每股未分配利润，可用于分红或再投资。
- `retainedps`：每股留存收益 = 盈余公积 + 未分配利润。

三者合计加上股本（1元/股）等于 BPS。

### 单季度 EPS
`fina_indicator` 还提供单季度每股指标，以 `q_` 前缀标识，如 `q_eps`（每股收益单季度），用于分析季度环比变化。
