# 指数专题

> 覆盖各类指数（沪深/行业/概念）的基础信息、日线/分钟线行情、成分权重、技术因子等数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `ci_index_member` | 按三级分类提取中信行业成分，可提供某个分类的所有成分，也可按股票代码提取所属分类，参数灵活 | l1_code, l2_code, l3_code, ts_code |
| `ci_daily` | 获取中信行业指数日线行情 | ts_code, trade_date, start_date, end_date |
| `index_global` | 获取国际主要指数日线行情 | ts_code, trade_date, start_date, end_date |
| `index_dailybasic` | 目前只提供上证综指，深证成指，上证50，中证500，中小板指，创业板指的每日指标数据 | trade_date, ts_code, start_date, end_date |
| `idx_mins` | 获取交易所指数分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式 | ts_code, freq, start_date, end_date |
| `index_weekly` | 获取指数周线行情 | ts_code, trade_date, start_date, end_date |
| `index_basic` | 获取指数基础信息。 | ts_code, symbol, name, market |
| `rt_idx_min` | 获取交易所指数实时分钟数据，包括1~60min | freq, ts_code |
| `rt_idx_k` | 获取交易所指数实时日线行情，支持按代码或代码通配符一次性提取全部交易所指数实时日k线行情 | ts_code |
| `index_weight` | 获取各类指数成分和权重，**月度数据** ，建议输入参数里开始日期和结束日分别输入当月第一天和最后一天的日期。 | index_code, trade_date, start_date, end_date |
| `idx_factor_pro` | 获取指数每日技术面因子数据，用于跟踪指数当前走势情况，数据由Tushare社区自产，覆盖全历史；输出参数_bfq表示不复权描述中说明了因子的默认传参，如需要特殊参数或者更多因子可以联系管理员评估，指数包括大盘指数 申万行业指数 中信指数 | ts_code, start_date, end_date, trade_date |
| `index_daily` | 获取指数每日行情，还可以通过bar接口获取。由于服务器压力，目前规则是单次调取最多取8000行记录，可以设置start和end日期补全。指数行情也可以通过**通用行情接口**获取数据。本接口不包含申万行业指数行情数据。 | ts_code, trade_date, start_date, end_date |
| `index_monthly` | 获取指数月线行情,每月更新一次 | ts_code, trade_date, start_date, end_date |
| `daily_info` | 获取交易所股票交易统计，包括各板块明细 | trade_date, ts_code, exchange, start_date |
| `sz_daily_info` | 获取深圳市场每日交易概况 | trade_date, ts_code, start_date, end_date |
| `rt_sw_k` | 获取申万行业指数的最新截面数据 | ts_code, ts_code, name, trade_time |
| `index_classify` | 获取申万行业分类，可以获取申万2014年版本（28个一级分类，104个二级分类，227个三级分类）和2021年本版（31个一级分类，134个二级分类，346个三级分类）列表信息 | index_code, level, parent_code, src |
| `index_member_all` | 按三级分类提取申万行业成分，可提供某个分类的所有成分，也可按股票代码提取所属分类，参数灵活 | l1_code, l2_code, l3_code, ts_code |
| `sw_daily` | 获取申万行业日线行情（默认是申万2021版行情） | ts_code, trade_date, start_date, end_date |

## 接口详细参数

### `ci_index_member` 详细说明

**调用说明**: 用户需5000积分可调取；单次最大5000行，总量不限制
**数据说明**: 按三级分类提取中信行业成分

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| l1_code | str | N | 一级行业代码 |
| l2_code | str | N | 二级行业代码 |
| l3_code | str | N | 三级行业代码 |
| ts_code | str | N | 股票代码 |
| is_new | str | N | 是否最新（默认为"Y是"） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| l1_code | str | 一级行业代码 |
| l1_name | str | 一级行业名称 |
| l2_code | str | 二级行业代码 |
| l2_name | str | 二级行业名称 |
| l3_code | str | 三级行业代码 |
| l3_name | str | 三级行业名称 |
| ts_code | str | 成分股票代码 |
| name | str | 成分股票名称 |
| in_date | str | 纳入日期 |
| out_date | str | 剔除日期 |
| is_new | str | 是否最新Y是N否 |
