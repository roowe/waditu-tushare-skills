# 外汇与海外资产

> 覆盖海外外汇及差价合约（CFD）的基础信息和日线行情。数据来源于 FXCM 交易商，除传统外汇货币对外，还涵盖海外指数、大宗商品、贵金属、债券、加密货币等多资产类别。共 2 个接口。

## 核心概念

### 外汇货币对

基础外汇品种如 USDCNH（美元人民币对）等主流货币对，通过 `fx_obasic` 获取交易参数（最小/最大交易单位、点值、点差等）。

### 差价合约（CFD）资产分类

`fx_obasic` 的 classify 参数支持 7 大资产类别：

| 分类代码 | 类别名称 | 样例 |
|----------|----------|------|
| FX | 外汇货币对 | USDCNH（美元人民币对） |
| INDEX | 海外指数 | US30（道琼斯）、SPX500（标普500）、NAS100（纳斯达克100） |
| COMMODITY | 大宗商品 | SOYF（大豆） |
| METAL | 贵金属 | XAUUSD（黄金） |
| BUND | 国库债券 | Bund（长期欧元债券） |
| CRYPTO | 加密数字货币 | BTCUSD（比特币） |
| FX_BASKET | 外汇篮子 | USDOLLAR（美元指数） |

### bid/ask 双报价体系

`fx_daily` 日线行情采用买入价（bid）和卖出价（ask）双报价：
- **bid_open/bid_close/bid_high/bid_low**：买入侧 OHLC
- **ask_open/ask_close/ask_high/ask_low**：卖出侧 OHLC
- **tick_qty**：报价笔数

注意：交易日期为 GMT 时间，比北京时间晚一天。

## 接口一览

| 接口名 | 说明 | 关键参数 | 积分 |
|--------|------|----------|------|
| `fx_obasic` | 外汇/CFD 基础信息 | exchange, classify, ts_code | 2000 |
| `fx_daily` | 外汇/CFD 日线行情 | ts_code, trade_date, start_date, end_date | 2000 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 获取海外指数类 CFD 基础信息
df = pro.fx_obasic(exchange='FXCM', classify='INDEX',
                   fields='ts_code,name,min_unit,max_unit,pip,pip_cost')

# 获取黄金 CFD 行情
df = pro.fx_obasic(exchange='FXCM', classify='METAL')

# 美元人民币日线行情
df = pro.fx_daily(ts_code='USDCNH.FXCM', start_date='20190101', end_date='20190524')

# 比特币日线行情
df = pro.fx_daily(ts_code='BTCUSD.FXCM', start_date='20230101', end_date='20231231')
```
