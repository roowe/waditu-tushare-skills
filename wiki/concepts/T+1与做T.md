# T+1与做T

> A股实行T+1交割制度，当日买入的股票需到下一个交易日才能卖出，投资者通过持有底仓实现日内回转交易（做T）以变通这一限制。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| T+1交割制度 | T+1 Settlement | 当日买入的股票次日才能卖出，资金同样实行T+1交收 | - |
| 做T/日内回转 | Day Trading via Existing Position | 利用已有底仓，在日内高卖低买（正T）或低买高卖（反T）实现套利 | - |
| 正T | Long T | 先在低位买入，待价格上涨后卖出原有底仓部分 | - |
| 反T | Reverse T | 先在高位卖出底仓，待价格回落后买回同等数量 | - |
| 锁仓 | Hold Position | 持仓不动，不进行任何买卖操作 | - |
| 底仓 | Base Position | 做T所需预先持有的股票仓位，是日内回转的基础 | - |
| 成本摊薄 | Cost Averaging | 通过做T获取的差价利润降低持仓成本 | - |

## 详细说明

T+1交割制度是中国A股市场的基本交易规则之一，自1995年1月1日起实施。该制度规定投资者当日通过集中竞价或大宗交易买入的A股股票，必须等到下一个交易日（T+1日）才能卖出。这一制度的初衷是为了抑制过度投机，降低市场波动性，但同时也限制了投资者的日内交易灵活性。相比之下，成熟市场如美股普遍实行T+0交易制度。

"做T"是投资者在T+1制度约束下发展出的变通策略。其核心思路是利用手中已持有的底仓（即前一交易日或更早买入的股票），在同一天内进行买卖操作，从而实现类似T+0的日内回转效果。具体来说，"正T"适用于预判股价先跌后涨的情形：先在低位买入与底仓等量的股票，待股价上涨后再卖出底仓部分，赚取中间差价；"反T"则适用于预判股价先涨后跌的情形：先在高位卖出底仓，待股价回落后再买回同等数量。

做T的风险在于方向判断失误。如果正T买入后股价继续下跌，则会增加持仓成本；反T卖出后股价继续上涨，则踏空后续涨幅。因此，做T需要对个股的日内波动节奏有较为准确的判断，通常结合分时均线、成交量等技术指标进行决策。对于量化交易者而言，做T策略需要精确计算交易成本（佣金+印花税），确保每次回转的预期收益能够覆盖交易费用。
