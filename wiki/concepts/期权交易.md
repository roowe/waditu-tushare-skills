# 期权交易

> 覆盖期权合约信息、日线与分钟级行情数据。支持全市场期权品种，包括股票期权、指数期权、ETF期权、商品期权等。共 3 个接口。

## 核心概念

### 期权合约

`opt_basic` 获取期权合约基础信息（5000 积分以上），关键字段：
- **call_put**：期权类型 —— Call（认购期权）/ Put（认沽期权）
- **exercise_type**：行权方式 —— 美式（如豆粕期权）/ 欧式（如50ETF期权）
- **exercise_price**：行权价格
- **per_unit**：合约单位（每手对应的标的数量）
- **s_month**：结算月
- **maturity_date**：到期日
- **opt_code**：标的合约代码（如 OP+期货合约代码）

支持按交易所（SSE/SZSE/DCE/SHFE/CZCE/CFFEX）和合约类型筛选。

### 期权类型分类

按标的资产划分：
- **股票期权**：个股期权（如上交所个股期权）
- **指数期权**：沪深300、中证1000等指数期权
- **ETF期权**：50ETF、300ETF、500ETF期权等
- **商品期权**：豆粕、白糖、铜等商品期货期权

### 期权日线行情

`opt_daily` 提供期权日线行情（2000 积分以上），单次最大 15000 条。包含 OHLC、结算价、前结算价、成交量/额、持仓量。支持按交易日期或合约代码循环提取。

### 期权分钟行情

`opt_mins` 提供分钟级行情（需单独权限），支持 1/5/15/30/60min 五种频率，单次最大 8000 行。输出包含 OHLC、成交量/额、持仓量。

## 接口一览

| 接口名 | 说明 | 关键参数 | 积分/权限 |
|--------|------|----------|-----------|
| `opt_basic` | 期权合约信息 | exchange, call_put, ts_code, opt_code | 5000 |
| `opt_daily` | 期权日线行情 | ts_code, trade_date, exchange | 2000 |
| `opt_mins` | 期权分钟行情（1-60min） | ts_code, freq, start_date, end_date | 单独权限 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 获取某交易所期权合约
df = pro.opt_basic(exchange='DCE', fields='ts_code,name,exercise_type,list_date,delist_date')

# 某日期权日线行情
df = pro.opt_daily(trade_date='20181212')

# 期权分钟行情
df = pro.opt_mins(ts_code='10007976.SH', freq='1min',
                  start_date='2024-09-27 09:00:00', end_date='2024-09-27 19:00:00')
```
