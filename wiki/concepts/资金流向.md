# 资金流向
> 资金流向反映市场中不同规模资金的进出方向和力度，是判断主力意图和市场趋势的重要参考指标。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 小单 | Small Order | 成交额 < 5万元的订单，通常代表散户资金 | `moneyflow` |
| 中单 | Medium Order | 成交额 5万~20万元的订单，代表中小型机构或大户 | `moneyflow` |
| 大单 | Large Order | 成交额 20万~100万元的订单，代表主力机构资金 | `moneyflow` |
| 特大单 | Extra Large Order | 成交额 >= 100万元的订单，代表大型机构资金 | `moneyflow` |
| 主动买入 | Active Buy | 以卖方报价成交的订单，反映主动性买盘意愿 | `moneyflow` |
| 主动卖出 | Active Sell | 以买方报价成交的订单，反映主动性卖盘意愿 | `moneyflow` |
| 净流入 | Net Inflow | 买入金额减去卖出金额为正值，表示资金净流入 | `moneyflow` |
| 净流出 | Net Outflow | 买入金额减去卖出金额为负值，表示资金净流出 | `moneyflow` |
| 主力资金 | Main Force | 大单和特大单资金的统称，通常被认为具有方向指引作用 | `moneyflow` |
| 个股资金流向 | Stock Money Flow | 单只股票各类型资金的买卖量和买卖金额明细 | `moneyflow` |
| 大盘资金流向 | Market Money Flow | 整体市场的资金流向汇总，按上证和深证分别统计 | `moneyflow_mkt_dc` |
| 板块资金流向 | Sector Money Flow | 行业、概念、地域板块层面的资金流向数据 | `moneyflow_ind_dc` |

## 详细说明

### 订单分类标准

个股资金流向数据按照成交金额将订单分为四档：

| 类型 | 金额范围 | 对应资金群体 |
|---|---|---|
| 小单 | < 5万元 | 散户 |
| 中单 | 5万~20万元 | 中小机构/大户 |
| 大单 | 20万~100万元 | 主力机构 |
| 特大单 | >= 100万元 | 大型机构 |

数据基于**主动买卖单**统计：主动买入指以卖方报价成交的订单，主动卖出指以买方报价成交的订单。净流入 = 主动买入金额 - 主动卖出金额。

### 三个维度

**个股维度（moneyflow）**
- 数据起始：2010年
- 单次最大提取6000行
- 包含每只股票的小单/中单/大单/特大单的买卖量和买卖金额，以及净流入量/额

**大盘维度（moneyflow_mkt_dc）**
- 数据来源：东方财富
- 按上证和深证分别统计
- 包含上证/深证收盘价、涨跌幅，以及主力净流入、超大单/大单/中单/小单净流入及其占比

**板块维度（moneyflow_ind_dc）**
- 数据来源：东方财富
- 支持行业、概念、地域三种板块类型
- 包含板块涨跌幅、主力净流入及各档位净流入占比、主力净流入最大个股

### 解读与应用

资金流向数据的典型应用场景：

1. **确认趋势**：股价上涨伴随主力资金持续净流入，趋势可靠性更高
2. **发现背离**：股价创新高但主力资金净流出，可能预示趋势反转
3. **板块轮动**：通过板块资金流向判断当前市场热点方向
4. **个股筛选**：在热门板块中选择主力资金净流入最大的个股

需要注意：资金流向数据基于成交明细的统计估算，并非真实资金进出的精确记录，需结合其他指标综合判断。
