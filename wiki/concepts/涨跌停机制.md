# 涨跌停机制

> A股交易中，个股当日价格波动不得超过前一交易日收盘价上下一定百分比的价格限制制度。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 涨停 | Limit Up | 主板+10%、科创板/创业板+20%，当日价格触及上限 | `limit_list_d` (limit_type='U') |
| 跌停 | Limit Down | 主板-10%、科创板/创业板-20%，当日价格触及下限 | `limit_list_d` (limit_type='D') |
| 炸板 | Broken Limit | 封住涨停后盘中被打开，说明多空分歧加大 | `limit_list_d` (limit_type='Z') |
| 连板 | Consecutive Limit Up | 连续涨停天数，反映个股市场热度 | `limit_list_d` (limit_times) |
| 连板天梯 | Limit Step | 市场中各连板层级（2板、3板...N板）的股票数量分布 | `limit_step` |
| 封单金额 | Seal Order Amount | 以涨停价挂单买入的资金总量，衡量封板力度 | `limit_list_d` (fd_amount) |
| 封板时间 | First/Last Limit Time | 首次封板时间和最后封板时间，越早封板越强势 | `limit_list_d` (first_time/last_time) |
| 开板次数 | Open Times | 盘中涨停被打开的次数，0表示一字封板 | `limit_list_d` (open_times) |
| 涨停统计 | Up Stat | N/T格式，T个交易日内有N次涨停 | `limit_list_d` (up_stat) |
| 涨跌停价格 | Daily Limit Price | 根据前收盘价和涨跌停比例每日计算 | `stk_limit` |

## 详细说明

涨跌停制度是中国A股市场特有的价格稳定机制。主板股票的涨跌幅限制为前一交易日收盘价的+-10%，ST股票为+-5%；科创板（688开头）和创业板（300/301开头）自2020年8月24日起实行注册制，涨跌幅限制放宽至+-20%。北交所（8开头）的涨跌幅限制为+-30%。当股票价格触及涨停价且在该价位有成交时，称为涨停；触及跌停价且有成交时，称为跌停。

封单金额和封板时间是衡量涨停质量的重要指标。封单金额越大，说明买方力量越强，次日继续上涨的概率越高；一字板（开板次数为0）通常被认为是最强的涨停形态。炸板（limit_type='Z'）则说明多头力量不足，盘中一度封住涨停但又被卖盘打开，这类股票次日走势不确定性较大。

连板天梯是市场情绪的直观体现。通过 `limit_step` 接口可以获取每日各连板层级的股票列表，当市场中出现10板以上的"妖股"时，通常意味着市场处于强势阶段；反之，若最高连板仅为2-3板，则市场情绪偏弱。涨停统计字段 up_stat 以"N/T"格式记录了个股近期的涨停频率，是量化打板策略的重要参考指标。

## 相关接口字段速查

**limit_list_d** -- 涨跌停列表

| 字段 | 类型 | 说明 |
|---|---|---|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| industry | str | 所属行业 |
| close | float | 收盘价 |
| pct_chg | float | 涨跌幅 |
| fd_amount | float | 封单金额 |
| first_time | str | 首次封板时间 |
| last_time | str | 最后封板时间 |
| open_times | int | 炸板/开板次数 |
| up_stat | str | 涨停统计（N/T） |
| limit_times | int | 连板数 |
| limit | str | D跌停/U涨停/Z炸板 |

**limit_step** -- 连板天梯

| 字段 | 类型 | 说明 |
|---|---|---|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| trade_date | str | 交易日期 |
| nums | str | 连板次数 |

**stk_limit** -- 每日涨跌停价格

| 字段 | 类型 | 说明 |
|---|---|---|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| pre_close | float | 昨日收盘价 |
| up_limit | float | 涨停价 |
| down_limit | float | 跌停价 |
