# 大模型语料

> 面向大模型训练和金融 NLP 的专题语料数据，涵盖公告全文、董秘问答、券商研报、政策法规、多源新闻等 8 大文本源。所有接口均需单独权限（与积分无关）。

## 核心概念

### 上市公司全量公告

`anns_d` 提供上市公司公告元数据及 PDF 下载 URL，单次最大 2000 条，可按日期循环提取全量。输出字段包括公告日期、股票代码/名称、标题、原文下载链接。

### 上证e互动问答

`irm_qa_sh` 记录上交所 e互动平台的董秘问答文本（2023年6月起），单次最大 3000 行。每条记录包含投资者提问（q）和公司回复（a），适用于情绪分析和投资者关系研究。

### 深证互动易问答

`irm_qa_sz` 记录深交所互动易平台的董秘问答文本（2010年10月起），单次最大 3000 行。相比上证 e互动，额外提供 **industry**（涉及行业）标签字段。

### 券商研究报告

`research_report` 提供券商研报元数据（2017年起），每日更新两次，单次最大 1000 条。支持按研报类别（个股/行业）、券商名称、行业名称筛选，输出包含标题、摘要、作者、下载链接。

### 国家政策法规库

`npr`（National Policy Repository）收录国务院等部门公开披露的政策法规全文，覆盖 110 个主题分类。输出包含 HTML 正文（content_html）、发文字号、发文机关、主题分类。

### 新闻快讯

`news` 汇聚 9 大新闻源的实时短讯，覆盖 6 年以上历史。数据源包括：

| 来源 | src 标识 |
|------|---------|
| 新浪财经 | sina |
| 华尔街见闻 | wallstreetcn |
| 同花顺 | 10jqka |
| 东方财富 | eastmoney |
| 云财经 | yuncaijing |
| 凤凰新闻 | fenghuang |
| 金融界 | jinrongjie |
| 财联社 | cls |
| 第一财经 | yicai |

单次最大 1500 条，按时间范围循环提取。

### 新闻联播文字稿

`cctv_news` 提供新闻联播每日分段文字稿（2017年起），每个段落带独立标题，便于筛选和聚合分析。

### 新闻通讯（长篇）

`major_news` 覆盖新华网、财新网、第一财经等 9 个来源的长篇通讯，提供超过 8 年历史。单次最大 400 行，content 字段需在 fields 中显式指定。

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `anns_d` | 上市公司全量公告（含 PDF URL） | ts_code, ann_date, start_date, end_date |
| `irm_qa_sh` | 上证e互动董秘问答 | ts_code, trade_date, start_date, end_date |
| `irm_qa_sz` | 深证互动易董秘问答（含行业标签） | ts_code, trade_date, start_date, end_date |
| `research_report` | 券商研报（个股/行业） | trade_date, report_type, ts_code, inst_csname |
| `npr` | 国家政策法规库 | org, ptype, start_date, end_date |
| `news` | 多源新闻快讯 | src, start_date, end_date |
| `cctv_news` | 新闻联播文字稿 | date |
| `major_news` | 长篇新闻通讯 | src, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 某日上市公司公告
df = pro.anns_d(ann_date='20230621')

# 上证e互动问答
df = pro.irm_qa_sh(trade_date='20250212')

# 深证互动易问答（含行业标签）
df = pro.irm_qa_sz(trade_date='20250212')

# 券商研报
df = pro.research_report(trade_date='20260121')

# 国务院科技类政策
df = pro.npr(org='国务院', ptype='科技')

# 财联社快讯
df = pro.news(src='cls', start_date='2025-01-15 09:00:00', end_date='2025-01-15 22:00:00')

# 新闻联播文字稿
df = pro.cctv_news(date='20181211')

# 长篇新闻（需指定 content 字段）
df = pro.major_news(src='新浪财经', start_date='2018-11-21 00:00:00',
                    end_date='2018-11-22 00:00:00', fields='title,content')
```
