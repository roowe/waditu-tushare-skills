# 限售与解禁

> 限售股在锁定期结束后进入二级市场流通的过程，大量解禁可能对股价形成显著供给压力。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 限售股 | Restricted Shares | 锁定期内不能在二级市场交易的股票，需满足条件后方可流通 | `share_float` |
| 解禁日 | Unlock Date | 限售股锁定期届满、可上市流通的日期 | `share_float` (float_date) |
| 定增股份 | Private Placement Shares | 定向增发获得的股份，通常有12个月或36个月锁定期 | `share_float` (share_type) |
| 首发原股东限售 | IPO Original Shareholder Lock-up | 公司IPO时原股东持有的股份，通常锁定12-36个月 | `share_float` (share_type) |
| 股权分置限售 | Split Share Structure Lock-up | 股改时原非流通股转为有限售期流通股 | `share_float` (share_type) |
| 解禁对股价影响 | Unlock Impact on Price | 大量限售股解禁增加市场供给，可能造成短期卖压和股价下跌 | - |
| 流通股份 | Floating Shares | 本次解禁可流通的股份数量 | `share_float` (float_share) |
| 流通比例 | Float Ratio | 解禁股份占总股本的比例 | `share_float` (float_ratio) |
| 公告日期 | Announcement Date | 限售股解禁的公告发布日期 | `share_float` (ann_date) |

## 详细说明

限售股是指在特定锁定期内不能在二级市场自由交易的股票，主要包括以下几类：首发原股东限售股（IPO时老股东持有，锁定期通常12-36个月）、定向增发限售股（参与定增的投资者获配股份，锁定期通常12个月或36个月）、股权激励限售股（激励对象获授股份，分批解锁）。限售制度的设计初衷是防止短期内大量股票涌入市场造成剧烈波动，同时绑定股东利益与公司长期发展。

解禁是指限售股锁定期届满后转为流通股，可在二级市场自由交易。解禁本身并不意味着股东一定会卖出，但大量解禁会增加市场的潜在供给压力。从历史数据看，解禁对股价的影响因情况而异：如果解禁股份占总股本比例较高（如超过5%），且解禁股东有较强的减持动机（如财务投资者），则股价可能面临较大下行压力；反之，如果解禁比例较小且股东为长期战略投资者，则影响有限。`share_float` 接口的 float_ratio 字段直接给出了解禁股份占总股本的比例，是评估解禁压力的关键指标。

对于量化分析而言，提前跟踪解禁日程表非常重要。通过 `share_float` 接口可以按解禁日期（float_date）或公告日期（ann_date）查询未来即将解禁的股票列表，结合 float_share 和 float_ratio 评估解禁规模，再叠加 holder_name（解禁股东名称）判断减持意愿，从而构建解禁事件驱动策略或风险预警模型。

## 相关接口字段速查

**share_float** -- 限售股解禁

| 字段 | 类型 | 说明 |
|---|---|---|
| ts_code | str | 股票代码 |
| ann_date | str | 公告日期 |
| float_date | str | 解禁日期 |
| float_share | float | 流通股份（股） |
| float_ratio | float | 流通股份占总股本比率 |
| holder_name | str | 股东名称 |
| share_type | str | 股份类型（定增股份、首发原股东限售等） |

**接口示例**

```python
# 按公告日期查询解禁信息
df = pro.share_float(ann_date='20181220')

# 按解禁日期范围查询
df = pro.share_float(start_date='20240101', end_date='20240331')

# 查询特定股票的解禁历史
df = pro.share_float(ts_code='000998.SZ')
```
