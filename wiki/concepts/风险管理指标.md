# 风险管理指标
> 风险管理指标是量化投资中衡量组合风险暴露和策略质量的核心工具，包括波动率、VaR、最大回撤、夏普比率等。这些指标虽不由Tushare直接提供，但均需基于Tushare行情数据计算得出。

## 核心概念

| 概念 | 英文 | 说明 | 计算依赖 |
|------|------|------|---------|
| 波动率 | Volatility (σ) | 收益率的标准差，衡量价格波动的剧烈程度 | `daily`(pct_chg) |
| 年化波动率 | Annualized Volatility | 日波动率 × √252（交易日），便于跨周期比较 | `daily`(pct_chg) |
| VaR | Value at Risk | 给定置信水平下，组合在一定持有期内的最大可能损失 | `daily`(close) |
| 历史VaR | Historical VaR | 基于历史收益分布的经验分位数 | `daily`(pct_chg) |
| 参数VaR | Parametric VaR | 假设收益服从正态分布，μ - z×σ | `daily`(pct_chg) |
| CVaR/ES | Conditional VaR / Expected Shortfall | VaR超出部分的平均损失，衡量尾部风险 | `daily`(pct_chg) |
| 最大回撤 | Maximum Drawdown (MDD) | 从峰值到谷值的最大跌幅，衡量最坏情况 | `daily`(close) |
| 卡玛比率 | Calmar Ratio | 年化收益率 / 最大回撤，衡量风险调整后收益 | `daily`(close/pct_chg) |
| 夏普比率 | Sharpe Ratio | (Rp - Rf) / σp，每承担一单位总风险获得的超额收益 | `daily`(pct_chg) |
| 信息比率 | Information Ratio (IR) | (Rp - Rb) / σ(Rp-Rb)，主动管理能力衡量 | `daily`(pct_chg) |
| Sortino比率 | Sortino Ratio | 仅用下行波动率（负收益的标准差）替代总波动率 | `daily`(pct_chg) |
| Beta系数 | Beta (β) | 个股/组合收益对市场收益的敏感度 | `daily`(pct_chg), 指数pct_chg |
| 跟踪误差 | Tracking Error | 组合收益与基准收益之差的标准差 | `daily`(pct_chg), 指数pct_chg |
| 下行风险 | Downside Risk | 低于目标收益率（通常为0或无风险利率）的收益的标准差 | `daily`(pct_chg) |
| 历史波动率 | Historical Volatility | 过去N个交易日收益率的标准差 | `daily`(pct_chg) |
| 隐含波动率 | Implied Volatility | 从期权价格反推的市场对未来波动率的预期 | 期权数据 |

## 详细说明

### 1. 波动率系列

- **历史波动率**: 最基础的风险度量，取过去N日（通常20/60/252日）收益率的标准差
  - 年化: σ_annual = σ_daily × √252
  - 缺点: 对极端事件反应滞后

- **ATR (Average True Range)**: `stk_factor_pro` 中的 `atr` 字段直接提供
  - 考虑了跳空缺口的真实波动幅度
  - 比收盘价标准差更全面地反映波动

- **GARCH模型**: 广义自回归条件异方差模型
  - 捕捉波动率聚集效应（大波动之后往往跟着大波动）
  - 在VaR计算中优于简单历史模拟

### 2. VaR系列

- **历史模拟法**: 取历史收益分布的第5%分位数（95%置信度）
- **参数法**: VaR = μ - z_α × σ × √Δt，假设正态分布
- **蒙特卡洛模拟**: 基于假设分布生成大量情景
- **CVaR (ES)**: 比VaR更好地捕捉尾部风险，监管偏好指标

### 3. 回撤系列

- **最大回撤(MDD)**: 从历史最高点到后续最低点的最大跌幅
  ```
  MDD = max((Peak - Trough) / Peak)
  ```
- **平均回撤**: 所有回撤的均值
- **回撤持续期**: 从回撤开始到恢复到前高的天数
- **卡玛比率**: 年化收益/MDD，>3为优秀

### 4. 风险调整收益系列

- **夏普比率**: 最广泛使用的风险调整指标
  - SR > 1: 优秀; SR > 2: 卓越; SR < 0: 亏损
  - 局限: 对非正态分布的收益有偏差
- **Sortino比率**: 仅惩罚下行波动，对正偏分布更公平
- **信息比率**: 衡量主动管理能力，IR > 0.5为优秀

### 5. Tushare数据支持

- `daily` 接口的 `pct_chg` 用于计算大部分风险指标
- `stk_factor_pro` 的 `atr` 直接提供波动率指标
- `daily_basic` 的 `turnover_rate` 可辅助流动性风险评估
- 以上风险指标需要用户自行计算，Tushare不直接提供
