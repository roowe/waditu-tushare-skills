# 趋势类指标
> 判断股价运行方向及趋势强度的技术分析指标集合，数据来源于 `stk_factor_pro` 接口。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口/字段 |
|---|---|---|---|
| MA | Simple Moving Average | 简单移动平均，取N日收盘价算术均值；周期覆盖 5/10/20/30/60/90/250 日 | `ma_bfq_5` ~ `ma_bfq_250`，三种复权 `_bfq`/`_hfq`/`_qfq` |
| EMA | Exponential Moving Average | 指数移动平均，近期数据权重更高；周期同 MA | `ema_bfq_5` ~ `ema_bfq_250`，三种复权 |
| MACD | Moving Average Convergence Divergence | DIF=EMA(SHORT)-EMA(LONG)，DEA=EMA(DIF,M)，MACD柱=2*(DIF-DEA)；参数 SHORT=12/LONG=26/M=9 | `macd_dif_*`、`macd_dea_*`、`macd_*`（MACD柱） |
| BOLL | Bollinger Bands | 布林带上/中/下轨，中轨=MA(N)，带宽=P倍标准差；参数 N=20/P=2 | `boll_upper_*`、`boll_mid_*`、`boll_lower_*` |
| DMI | Directional Movement Index | 动向指标，含 ADX(趋势强度)/PDI(多方)/MDI(空方)/ADXR；参数 M1=14/M2=6 | `dmi_adx_*`、`dmi_pdi_*`、`dmi_mdi_*`、`dmi_adxr_*` |
| DMA | Different of Moving Average | 平行线差指标，DIF=MA(N1)-MA(N2)，DIFMA=MA(DIF,M)；参数 N1=10/N2=50/M=10 | `dfma_dif_*`、`dfma_difma_*` |
| TRIX | Triple Exponential Smoothing | 三重指数平滑平均线，TRIX=三重EMA变化率，TRMA=TRIX的M2日均值；参数 M1=12/M2=20 | `trix_*`、`trma_*` |
| 薛斯通道II | Xue's Channel II | 含四条通道线 td1~td4，描述价格长期波动包络；参数 N=102/M=7 | `xsii_td1_*` ~ `xsii_td4_*` |
| 唐安奇通道 | Turtle Channel (Donchian) | 海龟交易法核心通道，上轨=N日最高、下轨=N日最低、中轨=均值；参数 N=20 | `taq_up_*`、`taq_mid_*`、`taq_down_*` |
| 肯特纳通道 | Keltner Channel | 中轨=EMA(N)，通道宽度=ATR(M)倍数；参数 N=20/ATR=10 | `ktn_upper_*`、`ktn_mid_*`、`ktn_down_*` |
| 连涨天数 | Consecutive Up Days | 当前连续上涨交易日数 | `updays` |
| 连跌天数 | Consecutive Down Days | 当前连续下跌交易日数 | `downdays` |
| topdays | Top Range | 当前最高价是近多少周期内最高价的最大值 | `topdays` |
| lowdays | Low Range | 当前最低价是近多少周期内最低价的最小值 | `lowdays` |

## 详细说明

所有趋势类指标均通过 `stk_factor_pro` 接口获取，每个指标字段名后缀为复权类型：
- `_bfq` -- 不复权（原始价格计算）
- `_hfq` -- 后复权（最早价不变，向后调整）
- `_qfq` -- 前复权（最新价不变，向前调整）

**典型用法**：MA/EMA 用于判断均线多头/空头排列及支撑压力位；MACD 金叉/死叉作为趋势转折信号；BOLL 收窄预示波动率压缩后突破；DMI 中 ADX>25 表示趋势明确。

**接口参数**：
```python
# 获取单只股票某日全部技术因子
df = pro.stk_factor_pro(ts_code='000001.SZ', trade_date='20240101')
# 按日期范围批量获取（单次最大10000条）
df = pro.stk_factor_pro(start_date='20240101', end_date='20240131')
```

**通用行情均线**：`pro_bar` 接口也支持动态计算均线（`ma=[5,20,50]`），返回字段为 `ma_5`、`ma_v_5`（均量）等。
