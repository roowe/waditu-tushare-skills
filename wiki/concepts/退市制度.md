# 退市制度

> A股退市制度是资本市场"有进有出"健康生态的重要组成部分，涵盖交易类、财务类、规范类、重大违法类四种强制退市路径以及主动退市，退市整理期和退市后股份转让（老三板）构成完整的退市生命周期。

## 核心概念

| 概念 | 说明 | 关键接口/字段 |
|------|------|---------------|
| 强制退市 | 交易所依规终止股票上市，包括四大类 | `stock_basic`(list_status='D') |
| 交易类退市 | 连续20个交易日收盘价低于1元/市值低于3亿/股东人数少于400人 | - |
| 财务类退市 | *ST后首个年报仍不符合条件（净利润为负且营收<1亿/净资产为负/审计意见非标） | `st`(st_tpye='*ST') |
| 规范类退市 | 信息披露重大缺陷/半数以上董事无法保证年报真实性/重大违规 | - |
| 重大违法类退市 | 欺诈发行/重大信息披露违法/严重损害社会公共利益 | - |
| 主动退市 | 上市公司主动申请终止上市（私有化/合并等） | - |
| 退市整理期 | 退市决定后15个交易日进入整理期交易，涨跌幅限制10% | - |
| 退市股份转让（老三板） | 退市后股票转入全国中小企业股份转让系统（两网及退市公司板块） | - |
| ST风险警示 | 退市风险的前置信号 | `st` |
| 退市日期 | `stock_basic` 的 delist_date 字段记录退市时间 | `stock_basic`(delist_date) |
| 退市状态 | list_status='D' 标识已退市股票 | `stock_basic`(list_status='D') |

## 详细说明

### 退市制度改革历程

- **2001年**: 首次建立退市制度（PT制度→退市）
- **2012年**: 新增"1元退市"和"成交量退市"
- **2014年**: 重大违法强制退市
- **2018年**: 重大违法强制退市修订
- **2020年**: 全面深化退市制度改革（退市新规），简化退市流程、新增市值退市、优化财务退市指标
- 注册制改革后退市效率显著提升

### 四类强制退市标准（2020年退市新规）

**交易类**: 触及以下任一条件即退市，无*ST过渡期
- 连续20个交易日每日股票收盘价均低于1元
- 连续20个交易日每日股票总市值均低于3亿元（科创板/创业板也为3亿元）
- 连续20个交易日股东人数均少于400人
- 连续120个交易日累计股票成交量低于500万股（或200万股）

**财务类**: 先*ST，次年年报仍不达标退市
- 最近一个会计年度经审计的净利润为负值且营业收入低于1亿元（扣除非经常性损益前后孰低）
- 最近一个会计年度经审计的期末净资产为负值
- 最近一个会计年度财务报告被出具无法表示意见或否定意见的审计报告

**规范类**: 先*ST，限期内未整改退市
- 信息披露存在重大缺陷
- 半数以上董事无法保证半数以上董事无法保证年报或半年报真实、准确、完整
- 未在法定期限内披露年度报告或半年度报告

**重大违法类**: 直接退市
- 欺诈发行
- 重大信息披露违法
- 严重损害社会公共利益

### 退市整理期

- 退市决定作出后，进入15个交易日的退市整理期
- 整理期内股票简称加"退"标识（如"XX退"）
- 涨跌幅限制10%（不设涨跌幅限制的情形除外）
- 整理期结束后，股票被摘牌

### 退市后的股份转让

- 摘牌后45个交易日内转入全国中小企业股份转让系统（老三板）
- 股东可在老三板进行股份转让
- 符合重新上市条件的退市公司可申请重新上市

## 相关接口

**stock_basic** -- 股票列表（退市筛选）
- `list_status='D'` 筛选已退市股票
- `delist_date` 字段记录退市日期
- 积分要求: 2000积分

**st** -- ST风险警示
- `st_tpye` 包含 *ST（退市风险警示）类型
- `st_reason` 和 `st_explain` 提供具体原因
- 积分要求: 6000积分
