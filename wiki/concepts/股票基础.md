# 股票基础

> 覆盖股票基础信息、上市公司信息、行业分类、交易日历等基础数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `new_share` | 获取新股上市列表数据 | start_date, end_date |
| `stock_st` | 获取ST股票列表，可根据交易日期获取历史上每天的ST列表 | ts_code, trade_date, start_date, end_date |
| `st` | ST风险警示板股票列表 | ts_code, pub_date, imp_date, ts_code |
| `stock_company` | 获取上市公司基础信息，单次提取4500条，可以根据交易所分批提取 | ts_code, exchange |
| `stk_managers` | 获取上市公司管理层 | ts_code, ann_date, start_date, end_date |
| `trade_cal` | 获取各大交易所交易日历数据,默认提取的是上交所 | exchange, start_date, end_date, is_open |
| `bse_mapping` | 获取北交所股票代码变更后新旧代码映射表数据 | o_code, n_code |
| `stk_premarket` | 每日开盘前获取当日股票的股本情况，包括总股本和流通股本，涨跌停价格等。 | ts_code, trade_date, start_date, end_date |
| `stock_hsgt` | 获取沪深港通股票列表 | ts_code, trade_date, type, start_date |
| `stk_rewards` | 获取上市公司管理层薪酬和持股 | ts_code, end_date |
| `stock_basic` | 获取基础信息数据，包括股票代码、名称、上市日期、退市日期等 | ts_code, name, market, list_status |
| `bak_basic` | 获取备用基础列表，数据从2016年开始 | trade_date, ts_code |
| `namechange` | 历史名称变更记录 | ts_code, start_date, end_date |

## 接口详细参数

### `new_share` 详细说明

**调用说明**: 单次最大2000条，总量不限制。需要至少120积分。

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | str | N | 上网发行开始日期 |
| end_date | str | N | 上网发行结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS股票代码 |
| sub_code | str | 申购代码 |
| name | str | 名称 |
| ipo_date | str | 上网发行日期 |
| issue_date | str | 上市日期 |
| amount | float | 发行总量（万股） |
| market_amount | float | 上网发行总量（万股） |
| price | float | 发行价格 |
| pe | float | 市盈率 |
| limit_amount | float | 个人申购上限（万股） |
| funds | float | 募集资金（亿元） |
| ballot | float | 中签率 |

### `stock_st` 详细说明

**调用说明**: 3000积分起。每天上午9:20更新，单次请求最大返回1000行数据，可循环提取。
**数据说明**: 数据从20160101开始，太早历史无法补齐。

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（格式：YYYYMMDD） |
| start_date | str | N | 开始时间 |
| end_date | str | N | 结束时间 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| trade_date | str | 交易日期 |
| type | str | 类型 |
| type_name | str | 类型名称 |

### `st` 详细说明

**调用说明**: 单次最大1000行，可根据股票代码循环获取历史数据。需要6000积分。

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| pub_date | str | N | 发布日期 |
| imp_date | str | N | 实施日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| pub_date | str | 发布日期 |
| imp_date | str | 实施日期 |
| st_tpye | str | 类型 |
| st_reason | str | st变更原因 |
| st_explain | str | st变更详细原因 |

| manager | str | 总经理 |
| secretary | str | 董秘 |
| reg_capital | float | 注册资本(万元) |
| setup_date | str | 注册日期 |
| province | str | 所在省份 |
| city | str | 所在城市 |
| introduction | str | 公司介绍 |

| email | str | 电子邮件 |
| office | str | 办公室 |
| employees | int | 员工人数 |
| main_business | str | 主要业务及产品 |
| business_scope | str | 经营范围 |

| trade_date | str | N | 交易日期(YYYYMMDD格式) |
| start_date | str | N | 开始日期 |

| 名称 | 类型 | 描述 |
|------|------|------|

| ts_code | str | TS股票代码 |

| 名称 | 类型 | 描述 |
|------|------|------|

### `stock_basic` 详细说明

**调用说明**: 2000积分起，每分钟请求50次。建议保存到本地存储后使用。每次最多返回6000行数据。

| list_status | str | N | 上市状态 L上市 D退市 P暂停上市 G未交易，默认是L |
| exchange | str | N | 交易所 SSE上交所 SZSE深交所 BSE北交所 |
| is_hs | str | N | 是否沪深港通标的，N否 H沪股通 S深股通 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS代码 |
| symbol | str | 股票代码 |
| name | str | 股票名称 |
| area | str | 地域 |
| industry | str | 所属行业 |
| fullname | str | 股票全称 |
| enname | str | 英文全称 |
| cnspell | str | 拼音缩写 |
| market | str | 市场类型（主板/创业板/科创板/CDR） |
| exchange | str | 交易所代码 |
| curr_type | str | 交易货币 |
| list_status | str | 上市状态 L上市 D退市 G过会未交易 P暂停上市 |
| list_date | str | 上市日期 |
| delist_date | str | 退市日期 |
| is_hs | str | 是否沪深港通标的，N否 H沪股通 S深股通 |
