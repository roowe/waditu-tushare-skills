# 资产负债表
> 反映企业在某一时点的资产、负债和股东权益状况，是财务分析的基石。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 货币资金 | Cash | 现金及等价物 | `balancesheet.money_cap` |
| 应收票据 | Notes Receivable | 商业汇票等应收票据 | `balancesheet.notes_receiv` |
| 应收账款 | Accounts Receivable | 未收回的货款 | `balancesheet.accounts_receiv` |
| 预付款项 | Prepayment | 预付给供应商的款项 | `balancesheet.prepayment` |
| 存货 | Inventories | 原材料、在产品、产成品 | `balancesheet.inventories` |
| 流动资产合计 | Total Current Assets | 一年内可变现或耗用的资产 | `balancesheet.total_cur_assets` |
| 固定资产 | Fixed Assets | 房屋、机器等长期资产 | `balancesheet.fix_assets` |
| 在建工程 | CIP | 尚未完工的固定资产建设 | `balancesheet.cip` |
| 无形资产 | Intangible Assets | 专利、商标等 | `balancesheet.intan_assets` |
| 商誉 | Goodwill | 并购溢价形成的资产 | `balancesheet.goodwill` |
| 长期股权投资 | LT Equity Investment | 对联营/合营企业的长期投资 | `balancesheet.lt_eqt_invest` |

| 短期借款 | ST Borrowing | 一年内需偿还的借款 | `balancesheet.st_borr` |
| 应付账款 | Accounts Payable | 欠供应商的货款 | `balancesheet.acct_payable` |
| 预收款项 | Advance Receipts | 预收客户的款项 | `balancesheet.adv_receipts` |
| 流动负债合计 | Total Current Liabilities | 一年内需偿还的负债 | `balancesheet.total_cur_liab` |

| 股本 | Total Share | 期末总股本 | `balancesheet.total_share` |
| 资本公积 | Capital Reserve | 股本溢价等 | `balancesheet.cap_rese` |

| 股东权益合计 | Total Holder Equity | 含/不含少数股东权益两种口径 | `balancesheet.total_hldr_eqy_inc_min_int` |

| 合同负债 | Contract Liability | 预收但尚未履约的义务 | `balancesheet.contract_liab` |

## 详细说明

### 资产结构
资产分为流动资产和非流动资产：
- **流动资产**：货币资金（`money_cap`）、应收票据（`notes_receiv`）、应收账款（`accounts_receiv`）、预付款项（`prepayment`）、存货（`inventories`）等。
- **非流动资产**：固定资产（`fix_assets`）、在建工程（`cip`）、无形资产（`intan_assets`）、商誉（`goodwill`）、长期股权投资（`lt_eqt_invest`）等。
- `fina_indicator` 提供 `ca_to_assets`（流动资产/总资产）和 `nca_to_assets`（非流动资产/总资产）两个比率辅助分析资产结构。

### 金融资产分类
资产负债表中新金融工具准则下的资产分类：
- **交易性金融资产**：`trad_asset`，以公允价值计量且变动计入当期损益。
- **可供出售金融资产**：`fa_avail_for_sale`（旧准则）。
- **持有至到期投资**：`htm_invest`（旧准则）。
- **应收款项类投资**：`invest_as_receiv`。
- 新准则下：`cost_fin_assets`（以摊余成本计量）、`fair_value_fin_assets`（以公允价值计量且变动计入其他综合收益）。

### 负债结构
- **流动负债**：短期借款（`st_borr`）、应付账款（`acct_payable`）、预收款项（`adv_receipts`）、合同负债（`contract_liab`）等。
- **非流动负债**：长期借款（`lt_borr`）、应付债券（`bond_payable`）、递延所得税负债（`defer_tax_liab`）等。

### 股东权益构成
股东权益 = 股本 + 资本公积 + 盈余公积 + 未分配利润 - 库存股 + 其他。
- `total_hldr_eqy_exc_min_int`：不含少数股东权益。
- `total_hldr_eqy_inc_min_int`：含少数股东权益。

### 特殊项目
- **合同资产负债**：`contract_assets` 和 `contract_liab` 是新收入准则下的科目，替代部分应收和预收。
- **递延所得税**：`defer_tax_assets`（递延所得税资产）和 `defer_tax_liab`（递延所得税负债）。
