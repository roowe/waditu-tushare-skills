# 期货持仓

> 期货持仓排名数据（fut_holding）按期货公司会员维度展示每日成交量和持仓量的排名情况，是推断机构仓位变动和市场多空力量对比的重要依据。通过追踪各期货公司的持买仓量和持卖仓量变化，可以洞察主力资金的交易方向。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 持仓排名(fut_holding) | 按期货公司会员的每日成交量/持仓量排名 | `fut_holding` |
| 持买仓量(long_hld) | 期货公司会员持有的多头仓位总量 | `long_hld` |
| 持卖仓量(short_hld) | 期货公司会员持有的空头仓位总量 | `short_hld` |
| 多空持仓变化(long_chg/short_chg) | 持仓量的日度增减变化 | `long_chg`, `short_chg` |
| 成交量排名(vol/vol_chg) | 期货公司的成交量排名及变化 | `vol`, `vol_chg` |
| 期货公司会员(broker) | 交易所会员的期货公司名称 | `broker` |
| 推断机构仓位变动 | 通过会员持仓变化推测背后的机构资金方向 | 综合 long_hld, short_hld, long_chg, short_chg |

## 详细说明

### 数据结构与获取

`fut_holding` 接口提供每日各期货品种按期货公司会员维度的成交量、持买仓量、持卖仓量及其变化数据。需要至少 2000 积分调用。可通过交易日期（trade_date）、合约/产品代码（symbol）、交易所（exchange）等参数筛选。

```python
pro = ts.pro_api()
# 获取玉米期货某日持仓排名
df = pro.fut_holding(trade_date='20181113', symbol='C1905', exchange='DCE')
# 获取某品种汇总排名
df = pro.fut_holding(trade_date='20181113', symbol='C')
```

### 多空力量分析

通过对比各期货公司的持买仓量（long_hld）和持卖仓量（short_hld），可以判断市场多空力量的集中度。重点关注头部会员的持仓变化：若某头部多头会员大幅增仓（long_chg 显著为正），可能暗示机构看多信号；反之亦然。

### 成交量与持仓量的关系

成交量排名（vol/vol_chg）反映交易活跃度，持仓量排名反映仓位方向。二者结合分析更有意义：高成交量伴随大幅增仓表示新资金入场，高成交量伴随减仓表示资金离场。部分会员可能仅出现在成交量排名中而不在持仓排名中（短线交易为主）。

### 数据特点

需要注意的是，fut_holding 数据按期货公司会员维度统计，同一期货公司可能汇总了多个客户的仓位，因此不能直接等同于单个机构的持仓。不同会员之间的仓位转移（换月移仓、移仓换月）也可能导致持仓数据剧烈变动。结合 [[资金流向]] 分析可以获取更完整的市场图景。
