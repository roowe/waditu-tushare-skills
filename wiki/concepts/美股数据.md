# 美股数据

> 覆盖美股基础信息、日线行情、分钟线、财务数据、分红拆股等数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `us_tradecal` | 获取美股交易日历信息 | start_date, end_date, is_open |
| `us_income` | 获取美股上市公司财务利润表数据（目前只覆盖主要美股和中概股） | ts_code, period, ind_name, report_type |
| `us_basic` | 获取美股列表信息 | ts_code, classify, offset, limit |
| `us_adjfactor` | 获取美股每日复权因子数据，在每天美股收盘后滚动刷新 | ts_code, trade_date, start_date, end_date |
| `us_daily_adj` | 获取美股复权行情，支持美股全市场股票，提供股本、市值、复权因子和成交信息等多个数据指标 | ts_code, trade_date, start_date, end_date |
| `us_daily` | 获取美股行情（未复权），包括全部股票全历史行情，以及重要的市场和估值指标 | ts_code, trade_date, start_date, end_date |
| `us_cashflow` | 获取美股上市公司现金流量表数据（目前只覆盖主要美股和中概股） | ts_code, period, ind_name, report_type |
| `us_fina_indicator` | 获取美股上市公司财务指标数据，目前只覆盖主要美股和中概股。为避免服务器压力，现阶段每次请求最多返回200条记录，可通过设置日期多次请求获取更多数据。 | ts_code, period, report_type, start_date |
| `us_balancesheet` | 获取美股上市公司资产负债表（目前只覆盖主要美股和中概股） | ts_code, period, ind_name, report_type |

## 接口详细参数

### `us_tradecal` 详细说明

**调用说明**: 单次最大6000条，可根据日期阶段获取
**数据说明**: 美股交易日历信息

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| is_open | str | N | 是否交易 0：休市、1：交易 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| cal_date | str | 日历日期 |
| is_open | int | 是否交易 '0'休市 '1'交易 |
| pretrade_date | str | 上一个交易日 |

---

### `us_income` 详细说明

**调用说明**: 需单独开权限或15000积分；单次请求最大返回10000行数据，可循环提取
**数据说明**: 美股上市公司财务利润表数据（目前只覆盖主要美股和中概股），按单只股票获取其历史数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| period | str | N | 报告期（格式：YYYYMMDD，每个季度最后一天的日期，如20241231） |
| ind_name | str | N | 指标名(如：新增借款） |
| report_type | str | N | 报告期类型(Q1一季报Q2半年报Q3三季报Q4年报) |
| start_date | str | N | 报告期开始时间（格式：YYYYMMDD） |
| end_date | str | N | 报告结束始时间（格式：YYYYMMDD） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| end_date | str | 报告期 |
| ind_type | str | 报告期类型(Q1一季报Q2半年报Q3三季报Q4年报) |
| name | str | 股票名称 |
| ind_name | str | 财务科目名称 |
| ind_value | float | 财务科目值 |
| report_type | str | 报告类型 |

---

### `us_basic` 详细说明

**调用说明**: 120积分可试用，5000积分有正式权限；单次最大6000条，可分页提取
**数据说明**: 美股列表信息

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码（如 AAPL） |
| classify | str | N | 股票分类 ADR/GDR/EQ |
| offset | str | N | 开始行数 |
| limit | str | N | 每页最大行数 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 美股代码 |
| name | str | 中文名称 |
| enname | str | 英文名称 |
| classify | str | 分类ADR/GDR/EQ |
| list_date | str | 上市日期 |
| delist_date | str | 退市日期 |

---

### `us_adjfactor` 详细说明

**调用说明**: 开通美股日线权限后自动获取权限；单次最大15000行数据，可根据日期循环
**数据说明**: 美股每日复权因子数据，每天美股收盘后滚动刷新

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（格式：YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| exchange | str | 交易所 |
| cum_adjfactor | float | 累计复权因子 |
| close_price | float | 收盘价 |
