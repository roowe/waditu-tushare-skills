# 注册制与发行制度
> 注册制是股票发行审核制度的重要变革，从核准制转向以信息披露为核心的注册制，显著改变了A股IPO的定价机制、上市节奏和市场结构。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|-------------|
| 核准制 | 2001-2019年A股主流发行制度，证监会实质审核IPO申请，关注企业盈利能力和发展前景 | - |
| 注册制 | 以信息披露为核心，交易所审核+证监会注册，强调将投资价值判断交给市场 | - |
| 科创板试点注册制 | 2019年7月22日科创板首批上市，A股首次实行注册制 | - |
| 创业板注册制 | 2020年8月24日创业板注册制首批上市，涨跌幅限制放宽至20% | - |
| 全面注册制 | 2023年2月17日全面实行股票发行注册制，主板也纳入注册制 | - |
| 询价制 | 机构投资者参与IPO定价，通过初步询价确定发行价格区间 | - |
| 绿鞋机制 | 超额配售选择权，主承销商可额外发行不超过15%的股份用于稳定上市初期股价 | - |
| 战略配售 | 向战略投资者定向配售部分新股，通常有锁定期 | - |
| 网上网下配售 | 网下（机构投资者）和网上（个人投资者）两种申购渠道 | - |
| 中签率 | 网上申购中签比率，反映市场对IPO的热度 | `new_share`(ballot) |
| 发行价格 | IPO定价，询价制下由机构报价决定 | `new_share`(price) |
| 发行市盈率 | 发行价格/每股收益，是衡量IPO定价高低的核心指标 | `new_share`(pe) |
| 募集资金 | IPO实际募集的资金总额 | `new_share`(funds) |
| 保荐机构跟投 | 注册制下保荐机构须以自有资金参与战略配售，跟投比例2%-5% | - |

## 详细说明

### 注册制改革历程

- **2019年7月**: 科创板试点注册制（688开头），取消盈利门槛，允许未盈利/特殊股权结构/红筹企业上市
- **2020年8月**: 创业板推广注册制（300/301开头），存量市场改革，涨跌幅从10%放宽至20%
- **2021年11月**: 北交所开市（8开头），注册制+涨跌幅30%
- **2023年2月**: 全面注册制，主板也实行注册制，但涨跌幅维持10%

### IPO定价机制

**询价制（注册制下）**:

1. **初步询价**: 机构投资者提交报价和拟申购数量
2. **剔除最高报价**: 通常剔除不超过所有配售对象拟申购总量10%的最高报价
3. **确定发行价格**: 计算剩余报价的中位数和加权平均数，确定发行价格
4. **科创板/创业板特殊规则**: 发行价可超过"四个值"孰低值（剔除最高报价后网下投资者报价的中位数和加权平均数、剔除最高报价后公募/社保/养老金等报价的中位数和加权平均数）

**固定价格法（核准制下主板常用）**: 直接确定发行价格

### 绿鞋机制（超额配售选择权）

- 发行人在IPO时授权主承销商超额配售不超过15%的股份
- 上市后30个自然日内，若股价**高于发行价**，主承销商行使超额配售权，从发行人处额外购买股份
- 若股价**跌破发行价**，主承销商从二级市场买入股票以支撑价格
- 绿鞋机制有助于稳定上市初期股价

- `new_share` 接口提供 IPO 新股上市数据
