# 偿债能力
> 衡量企业偿还到期债务（含短期和长期）的能力，评估财务风险和安全性。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 流动比率 | Current Ratio | 流动资产 / 流动负债，衡量短期偿债能力 | `fina_indicator.current_ratio` |
| 速动比率 | Quick Ratio | (流动资产 - 存货) / 流动负债 | `fina_indicator.quick_ratio` |
| 现金比率 | Cash Ratio | 保守速动比率，货币资金 / 流动负债 | `fina_indicator.cash_ratio` |
| 资产负债率 | Debt to Assets | 负债合计 / 资产总计 | `fina_indicator.debt_to_assets` |
| 权益乘数 | Assets to Equity | 总资产 / 股东权益，反映杠杆水平 | `fina_indicator.assets_to_eqt` |
| 产权比率 | Debt to Equity | 负债合计 / 股东权益 | `fina_indicator.debt_to_eqt` |
| 已获利息倍数 | EBIT to Interest | EBIT / 利息费用，衡量付息能力 | `fina_indicator.ebit_to_interest` |
| 净债务 | Net Debt | 有息债务减去货币资金 | `fina_indicator.netdebt` |
| 营运资金 | Working Capital | 流动资产 - 流动负债 | `fina_indicator.working_capital` |
| 有形资产/负债 | Tangible Asset to Debt | 有形资产对负债的覆盖程度 | `fina_indicator.tangibleasset_to_debt` |
| 经营现金流/流动负债 | OCF to Short Debt | 经营现金流对短期负债的覆盖 | `fina_indicator.ocf_to_shortdebt` |
| 经营现金流/负债合计 | OCF to Debt | 经营现金流对全部负债的覆盖 | `fina_indicator.ocf_to_debt` |

## 详细说明

### 短期偿债能力
三个递进口径的比率：
1. **流动比率**（`current_ratio`）：流动资产/流动负债，一般 > 2 认为较安全。但行业差异大。
2. **速动比率**（`quick_ratio`）：扣除存货后的流动资产/流动负债，一般 > 1 为宜。存货变现速度慢，扣除后更保守。
3. **现金比率**（`cash_ratio`）：最保守口径，仅考虑变现能力最强的资产。

### 长期偿债能力
- **资产负债率**（`debt_to_assets`）：最常用的杠杆指标。> 70% 通常被认为风险较高，但金融行业例外。
- **产权比率**（`debt_to_eqt`）：与资产负债率互为补充，反映债权人与股东权益的对比关系。
- **权益乘数**（`assets_to_eqt`）：资产负债率的倒数关系，用于 [[杜邦分析]]。

### 利息保障能力
- **已获利息倍数**（`ebit_to_interest`）：EBIT / 利息费用。倍数越高，付息能力越强，一般 > 3 较安全。
- **EBITDA/负债**（`ebitda_to_debt`）：从现金流角度衡量偿债能力。

### 债务结构分析
`fina_indicator` 提供债务结构的辅助分析：
- `currentdebt_to_debt`：流动负债/负债合计，反映短期偿债压力。
- `longdeb_to_debt`：非流动负债/负债合计。
- `int_to_talcap`：带息债务/全部投入资本。
- `eqt_to_talcapital`：归母权益/全部投入资本。

### 资产覆盖与现金覆盖
- `tangibleasset_to_debt`：有形资产对负债的覆盖（排除商誉等难以变现的资产）。
- `ocf_to_shortdebt`：经营现金流对短期负债的覆盖，体现真实的还债能力。
- `ocf_to_debt`：经营现金流对全部负债的覆盖。
