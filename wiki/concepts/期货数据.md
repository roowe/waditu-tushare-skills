# 期货数据

> 覆盖期货合约基础信息、日线/分钟线行情、仓单、持仓、交割、结算等全链路数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `trade_cal` | 获取各大期货交易所交易日历数据 | exchange, start_date, end_date, is_open |
| `fut_wsr` | 获取仓单日报数据，了解各仓库/厂库的仓单变化 | trade_date, symbol, start_date, end_date |
| `index_daily` | 获取南华指数每日行情，指数行情也可以通过**通用行情接口**获取数据． | ts_code, trade_date, start_date, end_date |
| `ft_mins` | 获取全市场期货合约分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式，如果需要主力合约分钟，请先通过主力mapping接口（需要有至少2000积分）获取对应的合约代码后提取分钟。 | ts_code, freq, start_date, end_date |
| `fut_basic` | 获取期货合约列表数据 | exchange, fut_type, fut_code, list_date |
| `rt_fut_min` | 获取全市场期货合约实时分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK、 http Restful API和websocket三种方式，如果需要主力合约分钟，请先通过主力mapping接口获取对应的合约代码后提取分钟。 | ts_code, freq |
| `fut_daily` | 期货日线行情数据 | trade_date, ts_code, exchange, start_date |
| `fut_mapping` | 获取期货主力（或连续）合约与月合约映射数据 | ts_code, trade_date, start_date, end_date |
| `fut_weekly_detail` | 获取期货交易所主要品种每周交易统计信息，数据从2010年3月开始 | week, prd, start_week, end_week |
| `ft_limit` | 获取所有期货合约每天的涨跌停价格及最低保证金率，数据开始于2005年。 | ts_code, trade_date, start_date, end_date |
| `fut_weekly_monthly` | 期货周/月线行情(每日更新) | ts_code, trade_date, start_date, end_date |
| `fut_holding` | 获取每日成交持仓排名数据 | trade_date, symbol, start_date, end_date |
| `fut_settle` | 获取每日结算参数数据，包括交易和交割费率等 | trade_date, ts_code, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 trade_cal 接口
df = pro.trade_cal(start_date="20240101", end_date="20240131")
print(df.head())
```
