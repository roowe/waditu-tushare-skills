# 期货数据

> 覆盖期货合约基础信息、日线/分钟线行情、仓单、持仓、交割、结算等全链路数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `trade_cal` | 获取各大期货交易所交易日历数据 | exchange, start_date, end_date, is_open |
| `fut_wsr` | 获取仓单日报数据，了解各仓库/厂库的仓单变化 | trade_date, symbol, start_date, end_date |
| `index_daily` | 获取南华指数每日行情，指数行情也可以通过**通用行情接口**获取数据． | ts_code, trade_date, start_date, end_date |
| `ft_mins` | 获取全市场期货合约分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式，如果需要主力合约分钟，请先通过主力mapping接口（需要有至少2000积分）获取对应的合约代码后提取分钟。 | ts_code, freq, start_date, end_date |
| `fut_basic` | 获取期货合约列表数据 | exchange, fut_type, fut_code, list_date |
| `rt_fut_min` | 获取全市场期货合约实时分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK、 http Restful API和websocket三种方式，如果需要主力合约分钟，请先通过主力mapping接口获取对应的合约代码后提取分钟。 | ts_code, freq |
| `fut_daily` | 期货日线行情数据 | trade_date, ts_code, exchange, start_date |
| `fut_mapping` | 获取期货主力（或连续）合约与月合约映射数据 | ts_code, trade_date, start_date, end_date |
| `fut_weekly_detail` | 获取期货交易所主要品种每周交易统计信息，数据从2010年3月开始 | week, prd, start_week, end_week |
| `ft_limit` | 获取所有期货合约每天的涨跌停价格及最低保证金率，数据开始于2005年。 | ts_code, trade_date, start_date, end_date |
| `fut_weekly_monthly` | 期货周/月线行情(每日更新) | ts_code, trade_date, start_date, end_date |
| `fut_holding` | 获取每日成交持仓排名数据 | trade_date, symbol, start_date, end_date |
| `fut_settle` | 获取每日结算参数数据，包括交易和交割费率等 | trade_date, ts_code, start_date, end_date |

## 接口详细参数

### `trade_cal` 详细说明

**调用说明**: 需2000积分

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| exchange | str | N | 交易所 SHFE上期所 DCE大商所 CFFEX中金所 CZCE郑商所 INE上海国际能源交易所 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| is_open | int | N | 是否交易 0休市 1交易 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| exchange | str | 交易所 |
| cal_date | str | 日历日期 |
| is_open | int | 是否交易 0休市 1交易 |
| pretrade_date | str | 上一个交易日 |

### `fut_wsr` 详细说明

**调用说明**: 需2000积分；单次最大1000，总量不限制

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| trade_date | str | N | 交易日期 |
| symbol | str | N | 产品代码 |
| start_date | str | N | 开始日期(YYYYMMDD格式，下同) |
| end_date | str | N | 结束日期 |
| exchange | str | N | 交易所代码 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| symbol | str | 产品代码 |
| fut_name | str | 产品名称 |
| warehouse | str | 仓库名称 |
| wh_id | str | 仓库编号 |
| pre_vol | int | 昨日仓单量 |
| vol | int | 今日仓单量 |
| vol_chg | int | 增减量 |
| area | str | 地区 |
| year | str | 年度 |
| grade | str | 等级 |
| brand | str | 品牌 |
| place | str | 产地 |
| pd | int | 升贴水 |
| is_ct | str | 是否折算仓单 |
| unit | str | 单位 |
| exchange | str | 交易所 |

### `index_daily` 详细说明

**调用说明**: 需2000积分

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | N | 指数代码（南华期货指数以 .NH 结尾） |
| trade_date | str | N | 交易日期（YYYYMMDD格式，下同） |
| start_date | str | N | 开始日期 |
| end_date | None | N | 结束日期 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | TS指数代码 |
| trade_date | str | 交易日 |
| close | float | 收盘点位 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| pre_close | float | 昨日收盘点 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅 |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

### `ft_mins` 详细说明

**调用说明**: 120积分可调取2次查看数据，正式权限需参阅权限说明；单次最大8000行数据，可循环获取，提供超过10年历史分钟数据
**数据说明**: 支持1min/5min/15min/30min/60min行情

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，e.g.CU2310.SHF |
| freq | str | Y | 分钟频度（1min/5min/15min/30min/60min） |
| start_date | datetime | N | 开始日期 格式：2023-08-25 09:00:00 |
| end_date | datetime | N | 结束时间 格式：2023-08-25 19:00:00 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_time | str | 交易时间 |
| open | float | 开盘价（元） |
| close | float | 收盘价（元） |
| high | float | 最高价（元） |
| low | float | 最低价（元） |
| vol | int | 成交量（手） |
| amount | float | 成交金额（元） |
| oi | float | 持仓量（手） |

### `fut_basic` 详细说明

**调用说明**: 需2000积分；单次最大10000

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|

| fut_type | str | N | 合约类型（1普通合约 2主力与连续合约 默认取全部） |
| fut_code | str | N | 标准合约代码，如白银AG、AP鲜苹果等 |
| list_date | str | N | 上市开始日期(格式YYYYMMDD) |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 合约代码 |
| symbol | str | 交易标识 |
| exchange | str | 交易市场 |
| name | str | 中文简称 |
| fut_code | str | 合约产品代码 |
| multiplier | float | 合约乘数(只适用于国债期货、指数期货) |
| trade_unit | str | 交易计量单位 |
| per_unit | float | 交易单位(每手) |
| quote_unit | str | 报价单位 |
| quote_unit_desc | str | 最小报价单位说明 |
| d_mode_desc | str | 交割方式说明 |
| list_date | str | 上市日期 |
| delist_date | str | 最后交易日期 |
| d_month | str | 交割月份 |
| last_ddate | str | 最后交割日 |
| trade_time_desc | str | 交易时间说明 |

### `rt_fut_min` 详细说明

**调用说明**: 需单独开权限；每分钟可以请求500次，支持多个合约同时提取

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，e.g.CU2310.SHF，支持多个合约（逗号分隔） |
| freq | str | Y | 分钟频度（1MIN/5MIN/15MIN/30MIN/60MIN） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| code | str | 股票代码 |
| freq | str | 频度 |
| time | str | 交易时间 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | int | 成交量 |
| amount | float | 成交金额 |
| oi | float | 持仓量 |

### `fut_weekly_detail` 详细说明

**调用说明**: 600积分可调取，5000积分以上正常调取不受限制；单次最大获取4000行数据
**数据说明**: 数据从2010年3月开始，数据来源中国证监会

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| week | str | N | 周期（每年第几周，e.g. 202001表示2020第1周） |
| prd | str | N | 期货品种（支持多品种输入，逗号分隔） |
| start_week | str | N | 开始周期 |
| end_week | str | N | 结束周期 |
| exchange | str | N | 交易所 |
| fields | str | N | 提取的字段 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| exchange | str | 交易所代码 |
| prd | str | 期货品种代码 |
| name | str | 品种名称 |
| vol | int | 成交量（手） |
| vol_yoy | float | 同比增减（%） |
| amount | float | 成交金额（亿元） |
| amout_yoy | float | 同比增减（%） |
| cumvol | int | 年累计成交总量（手） |
| cumvol_yoy | float | 同比增减（%） |
| cumamt | float | 年累计成交金额（亿元） |
| cumamt_yoy | float | 同比增减（%） |
| open_interest | int | 持仓量（手） |
| interest_wow | float | 环比增减（%） |
| mc_close | float | 本周主力合约收盘价 |
| close_wow | float | 环比涨跌（%） |
| week | str | 周期 |
| week_date | str | 周日期 |

### `ft_limit` 详细说明

| trade_date | str | N | 交易日期（格式：YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| cont | str | N | 合约代码（例如：cont='CU') |
| exchange | str | N | 交易所代码（例如：exchange='DCE') |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | TS股票代码 |
| name | str | 合约名称 |
| up_limit | float | 涨停价 |
| down_limit | float | 跌停价 |
| m_ratio | float | 最低交易保证金率（%） |
| cont | str | 合约代码 |
| exchange | str | 交易所代码 |
