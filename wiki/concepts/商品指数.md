# 商品指数

> 南华商品指数是国内最权威的商品期货指数体系，覆盖综合指数和农产品、能化、黑色、金属、贵金属等分品类指数，以及各单品种指数。期货品种交易周报则从证监会维度提供品种级别的成交/持仓/价格周度统计，是跟踪商品市场整体景气度的重要工具。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|------|------|--------------|
| 南华商品指数 | 综合反映国内商品期货价格走势 | NHCI.NH（南华商品指数） |
| 分品类指数 | 农产品/能化/黑色/金属/有色金属/贵金属等 | NHAI, NHECI, NHFI, NHMI, NHNFI, NHPMI |
| 单品种指数 | 各期货品种对应的南华指数 | CU.NH（沪铜）、RB.NH（螺纹钢）等 |
| 期货品种交易周报 | 交易所主要品种每周交易统计 | `fut_weekly_detail` |
| 成交量同比环比 | 周度成交量同比变化和环比变化 | `vol_yoy`, `interest_wow` |
| 主力合约收盘价 | 本周主力合约收盘价及环比涨跌 | `mc_close`, `close_wow` |

## 详细说明

### 南华商品指数体系

南华期货指数是国内使用最广泛的商品指数体系，以 `.NH` 为后缀。指数体系包括：

**综合指数**：
- NHCI.NH -- 南华商品指数（综合）
- NHII.NH -- 南华工业品指数

**分品类指数**：
- NHAI.NH -- 南华农产品指数
- NHECI.NH -- 南华能化指数
- NHFI.NH -- 南华黑色指数
- NHMI.NH -- 南华金属指数
- NHNFI.NH -- 南华有色金属指数
- NHPMI.NH -- 南华贵金属指数

**单品种指数**（部分）：CU.NH（沪铜）、AL.NH（沪铝）、ZN.NH（沪锌）、AU.NH（沪黄金）、AG.NH（沪银）、RB.NH（螺纹钢）、I.NH（铁矿石）、J.NH（焦炭）、JM.NH（焦煤）、RU.NH（天胶）、M.NH（豆粕）、Y.NH（豆油）、C.NH（玉米）、SR.NH（白糖）、CF.NH（棉花）、TA.NH（PTA）、SC.NH（原油）等。

### 指数行情数据

`index_daily` 接口提供南华指数的日线行情数据（close, open, high, low, pct_chg, vol, amount），与股票指数行情格式一致。需要至少 2000 积分调用。

```python
pro = ts.pro_api()
# 获取南华沪铜指数
df = pro.index_daily(ts_code='CU.NH', start_date='20180101', end_date='20181201')
```

### 期货品种交易周报

`fut_weekly_detail` 接口提供证监会口径的各期货品种每周交易统计数据，数据从 2010 年 3 月开始。核心字段包括：

- **成交量**：vol（本周成交量）、vol_yoy（同比增减%）、cumvol（年累计成交量）
- **成交金额**：amount（本周成交额，亿元）、amout_yoy（同比增减%）
- **持仓量**：open_interest（持仓量，手）、interest_wow（环比增减%）
- **价格**：mc_close（本周主力合约收盘价）、close_wow（环比涨跌%）

周度数据通过 `week` 参数控制（如 202001 表示 2020 年第 1 周），支持按品种（prd）和交易所（exchange）筛选。
