# 杜邦分析
> 将 ROE 分解为净利率、资产周转率和权益乘数三者的乘积，揭示盈利来源的结构。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 杜邦分解 | DuPont Analysis | ROE = 净利率 x 周转率 x 权益乘数 | — |
| 净利率 | Net Profit Margin | 净利润 / 营业收入，衡量盈利能力 | `fina_indicator.netprofit_margin` |
| 总资产周转率 | Asset Turnover | 营业收入 / 总资产，衡量运营效率 | `fina_indicator.assets_turn` |
| 杜邦权益乘数 | Equity Multiplier (DuPont) | 总资产 / 股东权益，衡量财务杠杆 | `fina_indicator.dp_assets_to_eqt` |
| 杜邦ROA | ROA (DuPont) | 净利率 x 总资产周转率 | `fina_indicator.roa_dp` |
| 权益乘数 | Assets to Equity | 总资产 / 股东权益（通用口径） | `fina_indicator.assets_to_eqt` |
| 年化ROA | ROA Yearly | 年化总资产净利率 | `fina_indicator.roa_yearly` |

## 详细说明

### 三因素分解模型

杜邦分析的核心公式：

```
ROE = 净利率 x 总资产周转率 x 权益乘数
    = (净利润/营业收入) x (营业收入/总资产) x (总资产/股东权益)
```

三个因素分别代表：
1. **净利率**（`netprofit_margin`）：反映产品竞争力和成本控制能力。高净利率通常意味着强定价权或低成本优势。
2. **总资产周转率**（`assets_turn`）：反映资产使用效率。零售行业通常周转率高，重资产行业较低。
3. **权益乘数**（`dp_assets_to_eqt`）：反映财务杠杆水平。乘数越大，负债越多，风险也越高。

### 杜邦 ROA
`roa_dp` 是杜邦分解中的中间指标：ROA = 净利率 x 周转率。它剔除了杠杆因素，纯粹衡量资产运营效率。

### 与通用权益乘数的区别
- `dp_assets_to_eqt`：杜邦分析专用口径。
- `assets_to_eqt`：通用权益乘数口径。
两者计算逻辑一致，区别在于底层数据来源的细微差异。

### 实际应用
通过比较同行业公司的杜邦分解结果，可以判断：
- 高 ROE 是来自盈利能力（净利率）、运营效率（周转率）还是财务杠杆（权益乘数）。
- 杠杆驱动的 ROE 可持续性较弱，盈利或效率驱动的 ROE 更具持续性。

结合 `balancesheet` 接口可进一步分析资产和权益的具体构成。
