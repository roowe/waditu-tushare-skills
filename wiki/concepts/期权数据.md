# 期权数据

> 覆盖期权合约基础信息、日线行情、希腊字母等数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `opt_mins` | 获取全市场期权合约分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式。 | ts_code, freq, start_date, end_date |
| `opt_basic` | 获取期权合约信息 | ts_code, exchange, list_date, opt_code |
| `opt_daily` | 获取期权日线行情 | ts_code, trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 opt_mins 接口
df = pro.opt_mins(ts_code="159919.SZ", freq="5MIN", start_date="20240101")
print(df.head())
```
