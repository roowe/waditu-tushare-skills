# 盈利能力
> 衡量企业利用资源创造利润的效率，是判断公司经营质量的核心维度。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 净资产收益率 | ROE | 净利润 / 股东权益，衡量股东资本回报效率 | `fina_indicator.roe` |
| 加权平均ROE | ROE_WAA | 按时间加权的平均净资产收益率 | `fina_indicator.roe_waa` |
| 扣非ROE | ROE_DT | 扣除非经常损益后的净资产收益率 | `fina_indicator.roe_dt` |
| 年化ROE | ROE_Yearly | 将期间ROE年化处理以便比较 | `fina_indicator.roe_yearly` |
| 总资产报酬率 | ROA | 息税前利润 / 总资产，衡量全部资产盈利能力 | `fina_indicator.roa` |
| 投入资本回报率 | ROIC | 税后经营利润 / 投入资本 | `fina_indicator.roic` |
| 净利润率 | Net Profit Margin | 净利润 / 营业收入，反映收入转化为利润的能力 | `fina_indicator.netprofit_margin` |
| 毛利率 | Gross Profit Margin | 毛利 / 营业收入，反映产品基础盈利空间 | `fina_indicator.grossprofit_margin` |
| 息税前利润 | EBIT | 利润总额 + 利息费用，不受资本结构和税率影响 | `fina_indicator.ebit` / `income.ebit` |
| 息税折旧摊销前利润 | EBITDA | EBIT + 折旧 + 摊销，更接近经营现金创造能力 | `fina_indicator.ebitda` / `income.ebitda` |
| 营业利润率 | Operating Profit Ratio | 营业利润 / 营业总收入 | `fina_indicator.op_of_gr` |

## 详细说明

### ROE 体系
ROE 是巴菲特最看重的指标之一。tushare 提供多种 ROE 变体：
- `roe`：基础摊薄 ROE。
- `roe_waa`：加权平均 ROE，考虑增发等股权变动。
- `roe_dt`：扣除非经常损益，排除一次性收益/损失的干扰。
- `roe_yearly`：年化处理，方便跨期比较。

ROE 拆解可参考 [[杜邦分析]]。

### ROA 与 ROIC
- **ROA**（`roa`）：总资产报酬率，反映公司使用全部资产（含负债）创造利润的效率。另有 `npta`（总资产净利润）使用净利润口径。
- **ROIC**（`roic`）：投入资本回报率，分母仅包含股东权益和有息负债，排除无息经营负债，更精确地衡量资本使用效率。

### 利润率指标
- **净利润率**（`netprofit_margin`）：最终落到股东口袋的利润占收入的比重。
- **毛利率**（`grossprofit_margin`）：收入减去直接成本后的毛利占比，体现定价权和成本控制能力。
- **EBIT/EBITDA**：消除融资和税收差异后的盈利指标，适合跨公司、跨行业比较。在 `income` 接口中也直接提供（`income.ebit`、`income.ebitda`）。

### 费用率辅助分析
`fina_indicator` 还提供各项费用占收入的比率辅助分析：
- `expense_of_sales`：销售期间费用率。
- `cogs_of_sales`：销售成本率。
- `saleexp_to_gr`、`adminexp_of_gr`、`finaexp_of_gr`：三费各自占比。
