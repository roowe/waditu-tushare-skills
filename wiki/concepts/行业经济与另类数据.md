# 行业经济与另类数据

> 覆盖影视文娱行业（电影票房、剧本/电视剧备案）和台湾电子产业链月营收等另类数据。适用于影视行业景气度跟踪和半导体产业链研究。共 8 个接口。

## 核心概念

### 电影剧本备案

`film_record` 获取全国电影剧本备案公示数据（120 积分起），关键字段：
- **rec_no**：备案号
- **script_writer**：编剧
- **classified**：影片分类
- **rec_result**：备案结果
- 日期不连续，按公布批次更新，单次最大 500 条

### 电视剧备案

`teleplay_record` 获取 2009 年以来全国电视剧备案公示（600 积分起），包含丰富的制作信息：
- **classify**：题材（当代都市/近代革命等）
- **episodes**：集数
- **prod_cycle**：制作周期
- **content**：内容提要（长文本）
- **org**：报备机构
- 支持按月份、机构名称、剧名查询

### 台湾电子产业月营收

`tmt_twincome` 追踪台湾 TMT 电子产业 65 类产品的月度营收汇总，是半导体/PCB/LCD 产业链景气度的前瞻指标。单次最多获取 30 个月数据，支持按产品代码和日期范围查询。重点产品线：

| 代码 | 类别 | 代码 | 类别 |
|------|------|------|------|
| 8 | PCB | 46 | 半导体 |
| 28 | LCD面板 | 47 | 晶圆制造 |
| 52 | IC设计 | 48 | IC封测 |
| 1 | PC | 2 | NB（笔记本） |

### 台湾电子个股营收

`tmt_twincomedetail` 在汇总基础上提供个股级别的月度营收明细，覆盖台积电(2330)、鸿海(2317)、联发科(2454)等 1000+ 家台湾电子公司。支持按产品代码+公司代码精确查询。

### 影院日度票房

`bo_cinema` 提供单影院级别的日度票房数据（2018年9月起，500 积分），包含：
- **aud_count**：观众人数
- **att_ratio**：上座率
- **avg_price**：场均票价
- **p_pc**：场均人次

### 电影日/周/月度票房

三套接口覆盖不同时间粒度的票房数据：

| 接口 | 粒度 | 历史起点 | 关键字段 |
|------|------|----------|----------|
| `bo_daily` | 日度 | 2018年9月 | day_amount, avg_price, wom_index |
| `bo_weekly` | 周度 | 2008年 | week_amount, up_ratio |
| `bo_monthly` | 月度 | 2008年1月 | month_amount, m_ratio |

均包含 **wom_index**（口碑指数）和 **rank**（排名）。

## 接口一览

| 接口名 | 说明 | 关键参数 | 积分 |
|--------|------|----------|------|
| `film_record` | 电影剧本备案 | ann_date, start_date, end_date | 120 |
| `teleplay_record` | 电视剧备案公示 | report_date, org, name | 600 |
| `tmt_twincome` | 台湾电子产业月营收（汇总） | item, date, start_date, end_date | - |
| `tmt_twincomedetail` | 台湾电子个股月营收 | item, symbol, date | - |
| `bo_cinema` | 影院日度票房 | date | 500 |
| `bo_daily` | 电影日度票房 | date | 500 |
| `bo_weekly` | 电影周度票房 | date | 500 |
| `bo_monthly` | 电影月度票房 | date | 500 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 电影剧本备案
df = pro.film_record(start_date='20181014', end_date='20181214')

# 电视剧备案
df = pro.teleplay_record(report_date='201905')

# 台湾 PCB 产业链月营收
df = pro.tmt_twincome(item='8', start_date='20120101', end_date='20181010')

# 台积电(2330)半导体月营收
df = pro.tmt_twincomedetail(item='46', symbol='2330')

# 影院日度票房
df = pro.bo_cinema(date='20181014')

# 电影日度票房排名
df = pro.bo_daily(date='20181014')

# 电影周度票房
df = pro.bo_weekly(date='20181008')

# 电影月度票房
df = pro.bo_monthly(date='20180901')
```
