# 经济景气

> PMI（采购经理指数）和 GDP（国内生产总值）是衡量经济景气度的两大核心指标。PMI 是月度发布的先行指标，50 为扩张与收缩的分界线；GDP 是季度发布的综合指标，反映经济总量和增速。两者结合可以全面评估经济的运行状态。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |

| 新订单/新出口订单 | 内需和外需的订单情况 | `pmi010500`, `pmi010900` |
| 从业人员指数 | 就业景气度 | `pmi010800` |
| 原材料库存指数 | 库存周期的重要参考 | `pmi010700` |
| 非制造业PMI | 建筑业和服务业的商务活动指数 | `pmi020100` |
| 建筑业/服务业PMI | 非制造业的分行业数据 | `pmi020101`, `pmi020102` |

## 详细说明

### PMI 指标体系

PMI（采购经理指数）通过对采购经理的月度调查汇总而成，是经济运行的先行指标。50 为荣枯线：高于 50 表示经济扩张，低于 50 表示经济收缩。

**制造业 PMI**（`cn_pmi`）包含丰富的分项指标：
- **构成指数**：生产指数（pmi010400）、新订单指数（pmi010500）、供应商配送时间（pmi010600）、原材料库存（pmi010700）、从业人员（pmi010800）
- **其他指标**：新出口订单（pmi010900）、进口（pmi011000）、采购量（pmi011100）、购进价格（pmi011200）、出厂价格（pmi011300）、产成品库存（pmi011400）
- **企业规模**：大型企业（pmi010100）、中型企业（pmi010200）、小型企业（pmi010300）
- **分行业**：装备制造业（pmi011700）、高技术制造业（pmi011800）、基础原材料制造业（pmi011900）、消费品制造业（pmi012000）

**非制造业 PMI** 包含商务活动（pmi020100）、新订单（pmi020200）、投入品价格（pmi020300）、销售价格（pmi020400）、从业人员（pmi020500）等分项，并可按建筑业（pmi020101）和服务业（pmi020102）分别查看。

**综合 PMI**（pmi030000）为制造业和非制造业的加权综合产出指数。

### GDP 数据

`cn_gdp` 接口提供季度 GDP 数据，包括总量（gdp）、当季同比增速（gdp_yoy），以及三次产业的结构数据：
- **第一产业**（pi/pi_yoy）：农林牧渔业
- **第二产业**（si/si_yoy）：工业和建筑业
- **第三产业**（ti/ti_yoy）：服务业

数据从 1952 年开始，是中国最长周期的经济数据之一。

### PMI 与 GDP 的关系

PMI 是月度发布的先行指标，能更及时地反映经济变化趋势；GDP 是季度发布的综合指标，更全面但滞后。PMI 连续数月高于 50 通常预示 GDP 增速上行，反之亦然。结合 [[通胀指标]] 和 [[货币与社融]] 可以更完整地刻画经济周期。
