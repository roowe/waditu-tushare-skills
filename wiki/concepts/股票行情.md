# 股票行情

> 覆盖A股日线/分钟线/复权行情、实时行情、涨跌停价格等股票行情数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `stk_mins` | 获取A股分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式 | ts_code, freq, start_date, end_date |
| `daily` | 获取股票行情数据，或通过**通用行情接口**获取数据，包含了前后复权数据 | ts_code, trade_date, start_date, end_date |
| `stk_week_month_adj` | 股票周/月线行情(复权--每日更新) | ts_code, trade_date, start_date, end_date |
| `stk_weekly_monthly` | 股票周/月线行情(每日更新) | ts_code, trade_date, start_date, end_date |
| `weekly` | 获取A股周线行情，本接口每周最后一个交易日更新，如需要使用每天更新的周线数据，请使用日度更新的周线行情接口。 | ts_code, trade_date, start_date, end_date |
| `bak_daily` | 获取备用行情，包括特定的行情指标(数据从2017年中左右开始，早期有几天数据缺失，近期正常) | ts_code, trade_date, start_date, end_date |
| `adj_factor` | 本接口由Tushare自行生产，获取股票复权因子，可提取单只股票全部历史复权因子，也可以提取单日全部股票的复权因子。 | ts_code, trade_date, start_date, end_date |
| `pro_bar` | 复权行情 | ts_code, start_date, end_date, asset |
| `rt_min` | 获取全A股票实时分钟数据，包括1~60min | freq, ts_code |
| `rt_k` | 获取实时日k线行情，支持按股票代码及股票代码通配符一次性提取全部股票实时日k线行情 | ts_code |
| `monthly` | 获取A股月线数据 | ts_code, trade_date, start_date, end_date |
| `suspend_d` | 按日期方式获取股票每日停复牌信息 | ts_code, trade_date, start_date, end_date |
| `daily_basic` | 获取全部股票每日重要的基本面指标，可用于选股分析、报表展示等。单次请求最大返回6000条数据，可按日线循环提取全部历史。 | ts_code, trade_date, start_date, end_date |
| `stk_limit` | 获取全市场（包含A/B股和基金）每日涨跌停价格，包括涨停价格，跌停价格等，每个交易日8点40左右更新当日股票涨跌停价格。 | ts_code, trade_date, start_date, end_date |
| `hsgt_top10` | 获取沪股通、深股通每日前十大成交详细数据，每天18~20点之间完成当日更新 | ts_code, trade_date, start_date, end_date |
| `ggt_top10` | 获取港股通每日成交数据，其中包括沪市、深市详细数据，每天18~20点之间完成当日更新 | ts_code, trade_date, start_date, end_date |
| `ggt_daily` | 获取港股通每日成交信息，数据从2014年开始 | trade_date, start_date, end_date |
| `ggt_monthly` | 港股通每月成交信息，数据从2014年开始 | month, start_month, end_month |
| `pro_bar` | 目前整合了股票（未复权、前复权、后复权）、指数、数字货币、ETF基金、期货、期权的行情数据，未来还将整合包括外汇在内的所有交易行情数据，同时提供分钟数据。不同数据对应不同的积分要求，具体请参阅每类数据的文档说明。 | ts_code, start_date, end_date, asset |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 stk_mins 接口
df = pro.stk_mins(ts_code="159919.SZ", freq="5MIN", start_date="20240101")
print(df.head())
```
