# 港股数据

> 覆盖港股基础信息、日线行情、分钟线、财务指标、港股通资金流向等数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `hk_tradecal` | 获取交易日历 | start_date, end_date, is_open |
| `hk_mins` | 港股分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式 | ts_code, freq, start_date, end_date |
| `hk_income` | 获取港股上市公司财务利润表数据 | ts_code, period, ind_name, start_date |
| `hk_basic` | 获取港股列表信息 | ts_code, list_status |
| `hk_adjfactor` | 获取港股每日复权因子数据，每天滚动刷新 | ts_code, trade_date, start_date, end_date |
| `hk_daily_adj` | 获取港股复权行情，提供股票股本、市值和成交及换手多个数据指标 | ts_code, trade_date, start_date, end_date |
| `rt_hk_k` | 获取港股实时日k线行情，支持按股票代码及股票代码通配符一次性提取全部股票实时日k线行情 | ts_code |
| `hk_daily` | 获取港股每日增量和历史行情，每日18点左右更新当日数据 | ts_code, trade_date, start_date, end_date |
| `hk_cashflow` | 获取港股上市公司现金流量表数据 | ts_code, period, ind_name, start_date |
| `hk_fina_indicator` | 获取港股上市公司财务指标数据，为避免服务器压力，现阶段每次请求最多返回200条记录，可通过设置日期多次请求获取更多数据。 | ts_code, period, report_type, start_date |
| `hk_balancesheet` | 获取港股上市公司资产负债表 | ts_code, period, ind_name, start_date |

## 接口详细参数

### `hk_tradecal` 详细说明

**调用说明**: 2000积分可调取，单次最大2000条
**数据说明**: 港股交易日历数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| is_open | str | N | 是否交易 '0'休市 '1'交易 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| cal_date | str | 日历日期 |
| is_open | int | 是否交易 '0'休市 '1'交易 |
| pretrade_date | str | 上一个交易日 |

---

### `hk_mins` 详细说明

**调用说明**: 120积分可调取2次查看数据，正式权限需单独开通；单次最大8000行数据
**数据说明**: 港股分钟数据，支持1min/5min/15min/30min/60min行情

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，e.g.00001.HK |
| freq | str | Y | 分钟频度（1min/5min/15min/30min/60min） |
| start_date | datetime | N | 开始日期 格式：2023-03-13 09:00:00 |
| end_date | datetime | N | 结束时间 格式：2023-03-13 19:00:00 |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_time | str | 交易时间 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | int | 成交量 |
| amount | float | 成交金额 |

---

### `hk_income` 详细说明

**调用说明**: 需单独开权限或15000积分；单次请求最大返回10000行数据，可循环提取
**数据说明**: 港股上市公司财务利润表数据，按单只股票获取其历史数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| period | str | N | 报告期(格式：YYYYMMDD） |
| ind_name | str | N | 指标名（如：营业额） |
| start_date | str | N | 报告期开始日期（格式：YYYYMMDD） |
| end_date | str | N | 报告结束始日期（格式：YYYYMMDD） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| end_date | str | 报告期 |
| name | str | 股票名称 |
| ind_name | str | 财务科目名称 |
| ind_value | float | 财务科目值 |

### `hk_basic` 详细说明

**数据说明**: 港股列表信息

|------|------|------|------|
| ts_code | str | N | TS代码 |
| list_status | str | N | 上市状态 L上市 D退市 P暂停上市 ，默认L |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票简称 |
| fullname | str | 公司全称 |
| enname | str | 英文名称 |
| cn_spell | str | 拼音 |
| market | str | 市场类别 |
| list_status | str | 上市状态 |
| list_date | str | 上市日期 |
| delist_date | str | 退市日期 |
| trade_unit | float | 交易单位 |
| isin | str | ISIN代码 |
| curr_type | str | 货币代码 |

### `hk_daily_adj` 详细说明

| 名称 | 类型 | 描述 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| close | float | 收盘价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| pre_close | float | 昨收价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅 |
| vol | None | 成交量 |
| amount | float | 成交额 |
| vwap | float | 平均价 |
| adj_factor | float | 复权因子 |
| turnover_ratio | float | 换手率(基于总股本) |
| free_share | None | 流通股本 |
| total_share | None | 总股本 |
| free_mv | float | 流通市值 |
| total_mv | float | 总市值 |
