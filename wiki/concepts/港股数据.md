# 港股数据

> 覆盖港股基础信息、日线行情、分钟线、财务指标、港股通资金流向等数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `hk_tradecal` | 获取交易日历 | start_date, end_date, is_open |
| `hk_mins` | 港股分钟数据，支持1min/5min/15min/30min/60min行情，提供Python SDK和 http Restful API两种方式 | ts_code, freq, start_date, end_date |
| `hk_income` | 获取港股上市公司财务利润表数据 | ts_code, period, ind_name, start_date |
| `hk_basic` | 获取港股列表信息 | ts_code, list_status |
| `hk_adjfactor` | 获取港股每日复权因子数据，每天滚动刷新 | ts_code, trade_date, start_date, end_date |
| `hk_daily_adj` | 获取港股复权行情，提供股票股本、市值和成交及换手多个数据指标 | ts_code, trade_date, start_date, end_date |
| `rt_hk_k` | 获取港股实时日k线行情，支持按股票代码及股票代码通配符一次性提取全部股票实时日k线行情 | ts_code |
| `hk_daily` | 获取港股每日增量和历史行情，每日18点左右更新当日数据 | ts_code, trade_date, start_date, end_date |
| `hk_cashflow` | 获取港股上市公司现金流量表数据 | ts_code, period, ind_name, start_date |
| `hk_fina_indicator` | 获取港股上市公司财务指标数据，为避免服务器压力，现阶段每次请求最多返回200条记录，可通过设置日期多次请求获取更多数据。 | ts_code, period, report_type, start_date |
| `hk_balancesheet` | 获取港股上市公司资产负债表 | ts_code, period, ind_name, start_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 hk_tradecal 接口
df = pro.hk_tradecal(start_date="20240101", end_date="20240131")
print(df.head())
```
