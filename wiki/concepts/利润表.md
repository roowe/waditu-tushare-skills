# 利润表
> 反映企业在一定期间内的收入、成本、费用和利润构成，是评估经营成果的核心报表。

## 核心概念

| 概念 | 英文/缩写 | 说明 | 相关接口字段 |
|---|---|---|---|
| 营业总收入 | Total Revenue | 企业全部业务收入的总和 | `income.total_revenue` |
| 营业收入 | Revenue | 主营业务收入 | `income.revenue` |
| 营业成本 | Operating Cost | 与营业收入直接对应的成本 | `income.oper_cost` |
| 销售费用 | Selling Expense | 营销推广相关支出 | `income.sell_exp` |
| 管理费用 | Admin Expense | 日常管理运营支出 | `income.admin_exp` |
| 财务费用 | Financial Expense | 利息支出等融资成本 | `income.fin_exp` |
| 资产减值损失 | Asset Impairment Loss | 存货、应收等资产减值 | `income.assets_impair_loss` |
| 营业利润 | Operating Profit | 营业收入减去营业总成本后的利润 | `income.operate_profit` |
| 利润总额 | Total Profit | 营业利润加营业外收支 | `income.total_profit` |
| 净利润 | Net Income | 利润总额减去所得税 | `income.n_income` |
| 归母净利润 | Net Income Attr P | 归属于母公司股东的净利润 | `income.n_income_attr_p` |
| 少数股东损益 | Minority Gain | 子公司中少数股东享有的利润 | `income.minority_gain` |
| 综合收益总额 | Total Comprehensive Income | 净利润加其他综合收益 | `income.t_compr_income` |
| 研发费用 | R&D Expense | 研发活动支出 | `income.rd_exp` |
| 投资收益 | Investment Income | 对外投资取得的收益 | `income.invest_income` |
| 公允价值变动 | FV Value Chg Gain | 金融资产公允价值变动 | `income.fv_value_chg_gain` |
| 所得税费用 | Income Tax | 当期所得税费用 | `income.income_tax` |

## 详细说明

### 收入到净利润的推导路径

```
营业总收入 (total_revenue)
  - 营业总成本 (total_cogs)
    = 营业成本 (oper_cost)
    + 三费: 销售(sell_exp) + 管理(admin_exp) + 财务(fin_exp)
    + 资产减值损失 (assets_impair_loss)
    + 其他
  + 公允价值变动 (fv_value_chg_gain)
  + 投资收益 (invest_income)
  = 营业利润 (operate_profit)
  + 营业外收入 (non_oper_income)
  - 营业外支出 (non_oper_exp)
  = 利润总额 (total_profit)
  - 所得税 (income_tax)
  = 净利润 (n_income)
    = 归母净利润 (n_income_attr_p) + 少数股东损益 (minority_gain)
```

### 三费分析
三费（销售、管理、财务费用）是观察公司经营效率的重要窗口：
- `fina_indicator` 提供三费占营业总收入的比率：`saleexp_to_gr`、`adminexp_of_gr`、`finaexp_of_gr`。
- `expense_of_sales` 汇总了期间费用率。

### 利润质量分析
- **归母净利润**（`n_income_attr_p`）：投资者最关心的指标，剔除了少数股东权益。
- **扣非净利润**：`fina_indicator.profit_dedt` 扣除非经常损益后的净利润，反映主业盈利能力。
- **营业利润率**：`fina_indicator.op_of_gr` 反映主业经营效率。

### 综合收益
`t_compr_income` 包含净利润和其他综合收益（`oth_compr_income`），反映企业全部价值变动。
