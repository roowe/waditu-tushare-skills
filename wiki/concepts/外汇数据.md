# 外汇数据

> 提供主要货币对的汇率行情数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `fx_obasic` | 获取海外外汇基础信息，目前只有FXCM交易商的数据 | exchange, classify, ts_code |
| `fx_daily` | 获取外汇日线行情 | ts_code, trade_date, start_date, end_date |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 fx_obasic 接口
df = pro.fx_obasic(ts_code="159919.SZ")
print(df.head())
```
