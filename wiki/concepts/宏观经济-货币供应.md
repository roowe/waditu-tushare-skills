# 宏观经济-货币供应

> 覆盖M0、M1、M2等货币供应量数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `cn_m` | 获取货币供应量之月度数据 | m, start_m, end_m, fields |

## 接口详细参数

### `cn_m` 详细说明

**调用说明**: 需600积分；单次最大5000行，一次可提取全部数据
**数据说明**: 月度数据

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| m | str | N | 月度（202001表示2020年1月） |
| start_m | str | N | 开始月度 |
| end_m | str | N | 结束月度 |
| fields | str | N | 指定输出字段（e.g. fields='month,m0,m1,m2'） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| month | str | 月份YYYYMM |
| m0 | float | M0（亿元） |
| m0_yoy | float | M0同比（%） |
| m0_mom | float | M0环比（%） |
| m1 | float | M1（亿元） |
| m1_yoy | float | M1同比（%） |
| m1_mom | float | M1环比（%） |
| m2 | float | M2（亿元） |
| m2_yoy | float | M2同比（%） |
| m2_mom | float | M2环比（%） |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 cn_m 接口
df = pro.cn_m()
print(df.head())
```
