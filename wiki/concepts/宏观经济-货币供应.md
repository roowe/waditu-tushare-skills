# 宏观经济-货币供应

> 覆盖M0、M1、M2等货币供应量数据

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `cn_m` | 获取货币供应量之月度数据 | m, start_m, end_m, fields |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 调用 cn_m 接口
df = pro.cn_m()
print(df.head())
```
