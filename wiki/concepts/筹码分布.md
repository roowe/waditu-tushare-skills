# 筹码分布
> 筹码分布展示的是某只股票在不同价位的持仓占比，通过分析筹码的集中与分散程度，可以判断支撑位、阻力位和市场参与者的盈亏状态。

## 核心概念

| 概念 | 英文 | 说明 | 相关接口 |
|---|---|---|---|
| 筹码分布 | Chip Distribution | 各价位上持仓筹码的占比分布，反映全体持仓者的成本结构 | `cyq_chips` |
| 分位成本 | Percentile Cost | 按持仓筹码累计百分比计算的各分位对应成本价 | `cyq_perf` |
| 胜率 | Winner Rate | 当前股价高于持仓成本的比例，即获利盘占比 | `cyq_perf` |
| 加权平均成本 | Weighted Average Cost | 所有持仓筹码按数量加权的平均成本价 | `cyq_perf` |
| 支撑位 | Support Level | 筹码密集区域的下沿，股价下跌至此容易获得买盘支撑 | - |
| 阻力位 | Resistance Level | 筹码密集区域的上沿，股价上涨至此容易遭遇卖盘压力 | - |

## 详细说明

### 筹码分布的原理

筹码分布（也叫流通筹码分布或CYQ指标）是基于成交量分布的一种技术分析工具。其核心思想是：

- 每一笔成交都会产生买方和卖方两个方向的筹码转移
- 在某个价位成交越密集，该价位积累的筹码就越多
- 通过统计历史成交数据，可以估算出当前各价位的持仓占比

筹码分布图通常以价格为纵轴、持仓占比为横轴，形成类似山峰的分布形态。筹码峰表示该价位持仓集中，筹码谷表示该价位持仓稀疏。

### 分位成本解读

`cyq_perf` 接口提供五个关键分位成本：

| 分位 | 字段 | 含义 |
|---|---|---|
| 5%分位 | cost_5pct | 5%的筹码成本低于此价格（底部成本） |
| 15%分位 | cost_15pct | 15%的筹码成本低于此价格 |
| 50%分位 | cost_50pct | 50%的筹码成本低于此价格（中位数成本） |
| 85%分位 | cost_85pct | 85%的筹码成本低于此价格 |
| 95%分位 | cost_95pct | 95%的筹码成本低于此价格（顶部成本） |

### 胜率的含义

胜率（winner_rate）表示当前股价高于持仓成本的比例。例如胜率为30%，意味着当前价格下有30%的持仓者是盈利的，70%处于亏损状态。

- **高胜率（>80%）**：绝大多数持仓者盈利，获利回吐压力增大，需警惕回调
- **中等胜率（40%-60%）**：多空较为均衡，市场处于正常波动状态
- **低胜率（<20%）**：绝大多数持仓者亏损，割肉动力减弱，可能出现反弹

### 应用场景

1. **判断支撑阻力**：筹码密集区对应强支撑或强阻力
2. **识别主力吸筹**：低位筹码持续集中，上方筹码逐渐减少
3. **判断洗盘到位**：高位筹码向低位转移，筹码分布趋于单峰密集
4. **辅助止盈止损**：关注关键分位成本线作为参考

### 相关接口说明

**每日筹码分布（cyq_chips）**
- 提供每只股票各价位的持仓占比（price + percent）
- 数据从2018年开始，每天18~19点更新
- 单次最大2000条

**每日筹码及胜率（cyq_perf）**
- 提供每只股票的分位成本、加权平均成本和胜率
- 包含历史最低价和最高价作为参考
- 单次最大5000条
