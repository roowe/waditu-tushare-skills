# 黄金现货市场

> 覆盖上海黄金交易所（SGE）现货合约的基础信息和日线行情。包含黄金、白银、铂金等多个品种。共 2 个接口。

## 核心概念

### 上海黄金交易所（SGE）

上海黄金交易所是中国唯一的国家级黄金交易平台，提供现货实盘和延期交易两大类合约。通过 `sge_basic` 可一次提取全部合约基础信息。

### 现货合约

主要品种包括：

| 合约代码 | 品种名称 | 交易类型 |
|----------|----------|----------|
| Au99.95 | 黄金9995 | 现货实盘 |
| Au99.99 | 黄金9999 | 现货实盘 |
| Au100g | 100克金条 | 现货实盘 |
| iAu99.99 | 国际板黄金9999 | 现货实盘 |
| PGC30g | 熊猫金币30克 | 现货实盘 |
| Pt99.95 | 铂金9995 | 现货实盘 |

### 延期交易合约

| 合约代码 | 品种名称 | 特点 |
|----------|----------|------|
| Au(T+D) | 黄金延期 | 主力合约，保证金交易 |
| mAu(T+D) | 迷你黄金延期 | 小规格 Au(T+D) |
| Ag(T+D) | 白银延期 | 白银品种 |
| Au(T+N1) | 黄金T+N1 | 双月交收 |
| Au(T+N2) | 黄金T+N2 | 单月交收 |

### 合约基础信息

`sge_basic` 输出字段包含：
- **t_unit**：交易单位（克/手）
- **margin_rate**：保证金比例
- **liq_rate**：违约金比例（%）
- **min_vol / max_vol**：单笔报价量范围
- **trade_time**：交易时间（含日盘和夜盘）

### 现货日行情

`sge_daily` 提供日线行情（2000 积分以上），数据由当日 9:00-15:30 交易和前一日夜盘 20:00-2:30 数据构成，成交量和金额为双向计量。特色字段：
- **price_avg**：加权平均价（元/克）
- **settle_vol**：交收量
- **settle_dire**：持仓方向（如"空支付给多"）

## 接口一览

| 接口名 | 说明 | 关键参数 | 积分 |
|--------|------|----------|------|
| `sge_basic` | SGE 现货合约基础信息 | ts_code | 5000 |
| `sge_daily` | SGE 现货合约日线行情 | ts_code, trade_date, start_date, end_date | 2000 |

## 典型用法

```python
import tushare as ts
pro = ts.pro_api('your_token')

# 获取全部合约基础信息
df = pro.sge_basic()

# 查询单个合约
df = pro.sge_basic(ts_code='Au99.95')

# 某日全品种行情
df = pro.sge_daily(trade_date='20220311')

# 某合约指定日期行情
df = pro.sge_daily(ts_code='Au(T+D)', start_date='20220301', end_date='20220311',
                   fields='ts_code,close,open,vol')
```
