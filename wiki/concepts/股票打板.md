# 股票打板

> 覆盖涨停板、跌停板、连板、炸板等打板相关专题数据接口

## 接口一览

| 接口名 | 说明 | 关键参数 |
|--------|------|----------|
| `dc_hot` | 获取东方财富App热榜数据，包括A股市场、ETF基金、港股市场、美股市场等等，每日盘中提取4次，收盘后4次，最晚22点提取一次。 | trade_date, ts_code, market, hot_type |
| `dc_member` | 获取东方财富板块每日成分数据，可以根据概念板块代码和交易日期，获取历史成分 | ts_code, con_code, trade_date, start_date |
| `dc_index` | 获取东方财富每个交易日的概念板块数据，支持按日期查询 | ts_code, name, trade_date, start_date |
| `dc_daily` | 获取东财概念板块、行业指数板块、地域板块行情数据，历史数据开始于2020年 | ts_code, trade_date, start_date, end_date |
| `ths_hot` | 获取同花顺App热榜数据，包括热股、概念板块、ETF、可转债、港美股等等，每日盘中提取4次，收盘后4次，最晚22点提取一次。 | trade_date, ts_code, market, is_new |
| `ths_daily` | 获取同花顺板块指数行情 | ts_code, trade_date, start_date, end_date |
| `limit_list_ths` | 获取同花顺每日涨跌停榜单数据，历史数据从20231101开始提供，增量每天16点左右更新 | trade_date, ts_code, limit_type, market |
| `ths_member` | 获取同花顺概念板块成分列表 | ts_code, con_code |
| `ths_index` | 获取同花顺板块指数，包括概念、行业、特色指数。 | ts_code, exchange, type |
| `hm_list` | 获取游资分类名录信息 | name |
| `stk_auction` | 获取当日个股和ETF的集合竞价成交情况，每天9点25~29分之间可以获取当日的集合竞价成交数据 | ts_code, trade_date, start_date, end_date |
| `kpl_list` | 获取开盘啦涨停、跌停、炸板等榜单数据 | ts_code, trade_date, tag, start_date |
| `limit_cpt_list` | 获取每天涨停股票最多最强的概念板块，可以分析强势板块的轮动，判断资金动向 | trade_date, ts_code, start_date, end_date |
| `limit_step` | 获取每天连板个数晋级的股票，可以分析出每天连续涨停进阶个数，判断强势热度 | trade_date, ts_code, start_date, end_date |
| `limit_list_d` | 获取A股每日涨跌停、炸板数据情况，数据从2020年开始（不提供ST股票的统计） | trade_date, ts_code, limit_type, exchange |
| `hm_detail` | 获取每日游资交易明细，数据开始于2022年8。游资分类名录，请点击<a href="https://tushare.pro/document/2?doc_id=311">游资名录</a> | trade_date, ts_code, hm_name, start_date |
| `tdx_index` | 获取通达信板块基础信息，包括概念板块、行业、风格、地域等 | ts_code, trade_date, idx_type |
| `tdx_member` | 获取通达信各板块成分股信息 | ts_code, con_code, trade_date, start_date |
| `tdx_daily` | 获取通达信各板块行情，包括成交和估值等数据 | ts_code, trade_date, start_date, end_date |
| `kpl_concept_cons` | 获取开盘啦概念题材的成分股 | trade_date, ts_code, con_code |
| `top_inst` | 龙虎榜机构成交明细 | trade_date, ts_code |
| `top_list` | 龙虎榜每日交易明细 | trade_date, ts_code |

## 接口详细参数

### `dc_hot` 详细说明

**调用说明**: 单次最大2000条，可根据日期等参数循环获取全部数据。需8000积分。
**数据说明**: 每日盘中提取4次，收盘后4次，最晚22点提取一次。

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| trade_date | str | N | 交易日期 |
| ts_code | str | N | TS代码 |
| market | str | N | 类型(A股市场、ETF基金、港股市场、美股市场) |
| hot_type | str | N | 热点类型(人气榜、飙升榜) |
| is_new | str | N | 是否最新（默认Y，N为盘中和盘后阶段采集） |

**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| trade_date | str | 交易日期 |
| data_type | str | 数据类型 |
| ts_code | str | 股票代码 |
| ts_name | str | 股票名称 |
| rank | int | 排行或者热度 |
| pct_change | float | 涨跌幅% |
| current_price | float | 当前价 |
| rank_time | str | 排行榜获取时间 |

| end_date | str | N | 结束日期（YYYYMMDD格式） |
