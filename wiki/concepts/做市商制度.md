# 做市商制度
> 做市商制度是由具备资格的券商充当买卖双方的对手方，持续提供买卖双边报价以维持市场流动性的交易机制。A股市场的做市商制度主要应用于科创板、北交所和银行间债券市场。

## 核心概念

| 概念 | 说明 | 关键字段/接口 |
|---|---|---|
| 做市商 | 具备交易所批准资格的证券公司，承担持续报价义务 | - |
| 双边报价 | 做市商同时报出买入价(bid)和卖出价(ask)，赚取价差 | - |
| 买卖价差 | ask-bid的差额，是做市商的主要利润来源，也是隐性交易成本 | - |
| 报价义务 | 做市商须满足最短报价时间/最大报价价差/最小报价数量等要求 | - |
| 科创板做市 | 2022年10月证监会发布科创板做市规定，2023年起实施 | - |
| 北交所做市 | 北交所自开市起引入做市商制度，提升中小企业股票流动性 | - |
| 做市借券 | 做市商可通过证金公司借入证券用于做市，解决券源不足问题 | `slb_len_mm` |
| 转融通做市 | 通过转融通机制为做市商提供券源支持 | `slb_len_mm` |

## 详细说明

### A股做市商制度发展

- **2014年**: 银行间债券市场引入做市商制度
- **2022年10月**: 证监会发布《证券公司科创板股票做市交易业务试点规定》
- **2023年**: 科创板做市正式实施，14家券商获批首批做市资格
- **北交所**: 自2021年开市起就引入了做市制度

### 做市商运作机制

- **持续报价义务**: 做市商须在交易时间内持续提供买卖双边报价
- **报价价差限制**: 买卖价差不得超过交易所规定的上限
- **最小报价数量**: 每次报价的最低申报数量要求
- **库存管理**: 做市商需要管理做市库存头寸的风险
- **借券机制**: `slb_len_mm` 接口记录做市商通过转融通借入证券的数据

### 对市场的影响

- **提升流动性**: 做市商的持续报价为市场提供了基础流动性
- **降低波动**: 做市商在买卖之间平滑价格波动
- **价格发现**: 集中的双边报价有助于更有效的价格发现
- **对量化策略的影响**: 做市商存在时，市场微观结构改变，高频策略需调整

### Tushare数据支持

- `slb_len_mm` 接口提供做市借券交易汇总数据（注意：该接口标注为"停"状态）
- 字段: trade_date, ts_code, name, ope_inv(期初余量), lent_qnt(融出数量), cls_inv(期末余量), end_bal(期末余额)

## 相关接口

**slb_len_mm** -- 做市借券交易汇总（停）
- 积分要求: 2000积分
- 单次最大5000行
- 注意: 该接口可能已停更

| 字段 | 类型 | 说明 |
|---|---|---|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| ope_inv | float | 期初余量(万股) |
| lent_qnt | float | 融出数量(万股) |
| cls_inv | float | 期末余量(万股) |
| end_bal | float | 期末余额(万元) |
